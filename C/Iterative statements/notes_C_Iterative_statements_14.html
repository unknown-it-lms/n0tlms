Notes: C- Iterative statements

Iterative statements-1
​​​​​​​Iterative statements, or loops, in C allow you to execute a block of code repeatedly based on a condition. There are several types of loops in C, each serving different purposes depending on the scenario.
Need of iterations
Sometimes there are some actions which need to be performed repetitively. Instead of writing such tasks again and again, loops are used. The actions which need to be repeated are written in the loop.

Every iteration has following important components:
  • The starting point of iteration.
  • The terminating point of the iteration.
  • The step to be taken to move from starting point to end point.
  • The condition that decides whether to continue the iteration or not.
  • The actual task to be done in the iteration.

Based on this, there are following three types of loops in C programming:
  1. while
  2. do-while
  3. for

​​​​​​​Lets now study each of them in detail.
while Loop 
This loop executes till the time condition is true.​​​​​​​
SyntaxExplanation
while(condition)
{
    // Body of the loop
}
  • While loop first tests the condition. Only of the condition is true, the body of loop gets executed. If the condition is false, the body of loop is skipped and execution starts from next line after the loop.
  • After execution of the body, the loop condition gets tested again and the above step repeats.
ExampleExplanation
int i =1;
while(i<=5)
{
  printf("%d", i);
  i++;
}		output
1 2 3 4 5 

Here, first i is initialized to 1. Then the loop condition is tested. If true, the loop body gets executed and value of i gets printed. Now i is increased by 1 to 2 (The step of loop) and again the loop condition is tested. This continues as long as the loop condition is true. 
When i becomes 6, the loop condition is false, the loop breaks and continues with the first line after the loop.

Note that: 'while' loop first tests the condition. Only if the condition is true, the body of loop is executed else it is skipped.
do..while loop
Syntax
SyntaxExplanation
do 
{
  statement 1;
  statement 2;
  :
} while(condition);

This loop is similar to while loop, except for one difference. In while loop, first the condition is tested and only of it is true, the loop body gets executed. In do-while loop, first the loop body gets executed and then the condition is tested. Only if the condition is true, the loop body gets executed again., else the loop breaks.

Hence, the loop body gets executed at-least once in do-while. Whereas, in while loop, the loop body may not get executed even once, if the condition is false at the beginning itself.
ExampleExplanation
int i=1;
do
{
    printf("%d",i);
    i++;
}while(i<=5);		Output
1 2 3 4 5
Condition in this loop is post-tested. Hence this loop will execute at least once. There will be at-least one iteration of this loop , irrespective of what value a variable has.
for loop
For loop has following components:
  1. Initialization: Decides the starting point of iteration
  2. Condition: Decides whether to execute the loop-body or to break the loop
  3. Increment: Decides the step of iteration
  4. Loop body: Actual code to be executed if the condition is true.
SyntaxExplanation
for(initialization; condition; increment)
 {
   statement 1;
   statement 2;
   :
 }		Note that the initialization part is executed only once, at the start. Afterwards, it is not executed at all, in any case.

ExampleExplanation
for(int i=1; i<=5; i++)
{
  printf("%d",i);
}		output
1 2 3 4 5

In the above case, statements will be repeated till the time condition remain true.
When condition becomes false, loop terminates. When value of i becomes 6 , loop terminates. Hence repetitive task of printing the number is kept in the loop.
