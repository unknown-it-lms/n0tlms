Basics of C

C is a widely-used, general-purpose programming language, developed in the early 1970s by Dennis Ritchie at Bell Labs. C has influenced many other programming languages, including C++, C#, and Java. Hence, it is aptly called as the "Mother of all programming languages".

Keywords:
Keywords are reserved words in C that have special meaning to the compiler. They are used to perform specific tasks and cannot be used as identifiers (like variable names). There are total 32 keywords in standard C (as per ANSI C).
Note that all keyswords are in small case. They are used by the compiler only for some specific purpose. Hence, they can NOT be used by programmers in any other way by the programmer.

Variables

In C, variables are used to store data that can be manipulated during program execution. Each variable must be declared with a specific data type that determines the size and nature of the data it can hold. A variable is a name of the memory location. It is used to store data. Its value can be changed, and it can be reused many times. Here's a detailed overview of variables in C:

Key Aspects of Variables in C:

  1. Declaration: Before using a variable, it must be declared with a data type. The declaration informs the compiler of the variable's type and name.

  2. Initialization: Variables can be initialized with a value at the time of declaration. If not initialized, the variable contains a garbage value.

  3. Scope: The scope of a variable defines where it can be accessed in the code. Variables can be local (within a function or block) or global (accessible throughout the program).

  4. Lifetime: The lifetime of a variable refers to how long the variable exists in memory. This can be automatic (local variables), static (preserve value across function calls), or dynamic (allocated on the heap).

  5. Storage Classes: Storage classes in C define the scope, visibility, and lifetime of variables. Common storage classes include auto, register, static, and extern.

Operator precedence
Operator precedence in C determines the order in which operators are evaluated within an expression. Operators with higher precedence are evaluated before those with lower precedence. This concept is fundamental to ensuring that expressions are calculated correctly and predictably. 

For example:
Consider the expression: int x = 4 + 2 * 8;
Without any rule for operator precedence, one can evaluate the above expression in two ways.
  • First (4+2) is calculated and the result is then multiplied with 8, giving the result as 48.
  • Or (2*8) is calculated and the result is then added to 4, giving the result as 64
Operator precedence avoids such confusion, by clearly specifing the order in which operators should be evaluated in an expression. 

Following are the rules of operator precedence.
  • Higher Precedence First:  Operators with higher precedence are evaluated before operators with lower precedence in an expression.
  • Parentheses Override Precedence:  Parentheses () can be used to explicitly control the order of evaluation, overriding the default operator precedence. Expressions within parentheses are always evaluated first.
  • Associativity for Same Precedence:  When two or more operators with the same precedence appear in an expression, their associativity determines the order of evaluation (e.g., left-to-right or right-to-left).

    Built-in functions

    C Standard library functions or simply C Library functions are inbuilt functions in C programming.

    The prototype and data definitions of these functions are present in their respective header files. To use these functions we need to include the header file in our program. For example,

    If you want to use the printf() function, the header file <stdio.h> should be included.

    scanf and printf are standard library functions in C used for input and output operations, respectively. They are defined in the stdio.h header file and are essential for interacting with users and displaying information.

    printf

    The printf function is used to output formatted data to the standard output (usually the console). It allows you to print text, variables, and formatted data.

    Syntax:

    int printf(const char *format, ...);

    • format: A string that specifies how subsequent arguments are formatted and displayed. It can include format specifiers (e.g., %d, %f, %s) that indicate the type of data to be printed.
    • ...: Additional arguments corresponding to the format specifiers in the format string.

    Common Format Specifiers:

    • %d or %i : Integer
    • %f : Floating-point number
    • %c : Character
    • %s : String
    • %x : Hexadecimal integer
    • %p : Pointer address
    #include <stdio.h>

    int main() {
        int age = 25;
        float height = 5.9;
        char grade = 'A';
        char name[] = "John Doe";

        // Print formatted output
        printf("Name: %s\n", name);
        printf("Age: %d\n", age);
        printf("Height: %.1f\n", height);
        printf("Grade: %c\n", grade);

        return 0;
    }

    scanf

    The scanf function is used to read formatted input from the standard input (usually the keyboard). It allows you to capture user input and store it in variables.

    Syntax:

    int scanf(const char *format, ...);

    • format: A string that specifies the expected format of the input data. It includes format specifiers corresponding to the types of variables where input will be stored.
    • ...: Pointers to variables where the input data will be stored.

    Common Format Specifiers:

    • %d or %i : Integer
    • %f : Floating-point number
    • %c : Character
    • %s : String
    • %x : Hexadecimal integer
    #include <stdio.h>

    int main() {
        int age;
        float height;
        char name[50];

        // Prompt user for input
        printf("Enter your name: ");
        scanf("%49s", name);  // %49s to avoid buffer overflow

        printf("Enter your age: ");
        scanf("%d", &age);

        printf("Enter your height: ");
        scanf("%f", &height);

        // Print the captured input
        printf("Name: %s\n", name);
        printf("Age: %d\n", age);
        printf("Height: %.1f\n", height);

        return 0;
    }
