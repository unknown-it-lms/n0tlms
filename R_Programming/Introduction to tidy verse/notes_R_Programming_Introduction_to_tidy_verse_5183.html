Introduction to tidy verse

Tidyverse is a collection of R packages designed for data science and data analysis. It was created by Hadley Wickham and his team at RStudio. The packages in Tidyverse are built on top of R's base functionality, and they share a common philosophy and design style.

The main goal of Tidyverse is to provide a consistent framework for working with data. The packages within the Tidyverse have a common set of principles and syntax that make them easy to learn and use together. They also make it easy to perform common data analysis tasks, such as data cleaning, transformation, visualization, and modeling.

The core packages of Tidyverse include:

  1. ggplot2 - for data visualization
  2. dplyr - for data manipulation
  3. tidyr - for data tidying
  4. readr - for data input
  5. purrr - for functional programming
  6. tibble - for modern data frames

Tidyverse also includes other packages that complement these core packages, such as stringr for working with strings, lubridate for working with dates, and forcats for working with categorical variables.

Overall, Tidyverse is a powerful and intuitive tool for data analysis and visualization, and it has become a popular choice among data scientists and analysts who use R for their work.

dplyr

dplyr is a popular R package used for data manipulation and transformation tasks. It provides a grammar for data manipulation by offering a set of functions that can be combined to create powerful data processing pipelines.

Example:
# Load the dplyr library
library(dplyr)

# Create a sample data frame
df <- data.frame(
  name = c("Alice", "Bob", "Charlie", "David"),
  age = c(25, 30, 35, 40),
  gender = c("F", "M", "M", "M"),
  salary = c(50000, 60000, 70000, 80000)
)
# Use dplyr to manipulate the data
df %>%
  filter(gender == "M") %>% # select only males
  mutate(age_group = ifelse(age < 35, "Under 35", "35 and over")) %>% # create a new column based on age
  select(name, age_group, salary) # select only relevant columns

# Output:
#    name age_group salary
# 1   Bob Under 35  60000
# 2 Charlie 35 and over 70000
# 3  David 35 and over 80000

In the above example, we loaded the dplyr package, created a sample data frame df, and then used the %>% operator (also known as the pipe operator) to chain together a series of dplyr functions to manipulate the data.

First, we used filter() to select only rows where the gender column was "M". Then, we used mutate() to create a new column called age_group that classified individuals as either "Under 35" or "35 and over" based on their age. Finally, we used select() to choose only the name, age_group, and salary columns from the resulting data frame.

List of functions available in the dplyr package.

  • select(): Select columns from your dataset
  • filter(): Filter out certain rows that meet your criteria(s)
  • group_by(): Group different observations together such that the original dataset does not change. Only the way it is represented is changed in the form of a list
  • summarise(): Summarise any of the above functions
  • arrange(): Arrange your column data in ascending or descending order
  • join(): Perform left, right, full, and inner joins in R
  • mutate(): Create new columns by preserving the existing variables

Manipulating and processing data in r

R is a powerful language for manipulating and processing data. Here are some basic ways to manipulate and process data in R:

  1. Subsetting data: Use the subset() function to extract rows or columns from a data frame based on a condition. For example, to extract all rows where the age variable is greater than 25, you can use subset(data, age > 25).

  2. Filtering data: Use the filter() function from the dplyr package to extract rows from a data frame based on multiple conditions. For example, to extract all rows where the age variable is greater than 25 and the gender variable is "female", you can use
    filter(data, age > 25, gender == "female").

  3. Creating new variables: Use the mutate() function from the dplyr package to add new variables to a data frame based on existing variables. For example, to add a new variable income to a data frame based on the salary and hours_worked variables, you can use mutate(data, income = salary * hours_worked).

  4. Aggregating data: Use the group_by() and summarize() functions from the dplyr package to compute summary statistics for groups of observations. For example, to compute the mean salary for each gender in a data frame, you can use
    data %>%
    group_by(gender) %>%
    summarize(mean_salary = mean(salary))    .

  5. Reshaping data: Use the reshape2 package to reshape data from a wide format to a long format or vice versa. For example, to reshape a data frame from a wide format to a long format, you can use the melt() function.

  6. Sorting data: Use the arrange() function from the dplyr package to sort a data frame by one or more variables. For example, to sort a data frame by the age variable in ascending order, you can use arrange(data, age).

  7. Combining data: Use the merge() or join() functions from the dplyr package to combine two or more data frames into a single data frame. For example, to combine two data frames based on a common variable id, you can use
    ​​​​​​​merge(data1, data2, by = "id").

Creating accessing and sorting data frames

  1.  Creating a data frame: Use the data.frame() function to create a data frame from vectors or lists. For example, to create a data frame with three variables name, age, and gender, you can use data.frame(name = c("Alice", "Bob", "Charlie"), age = c(25, 30, 35), gender = c("female", "male", "male")).

  2. Accessing a data frame: Use the [ ] or [[ ]] operators to access rows or columns of a data frame. For example, to access the first row of a data frame, you can use data[1, ], and to access the age variable, you can use data$age or data[["age"]].

  3. Sorting a data frame: Use the order() function to sort a data frame by one or more variables. For example, to sort a data frame data by the age variable in ascending order, you can use data[order(data$age), ]. To sort by multiple variables, use order(data$age, data$name).

    # Create a sample data frame
    df <- data.frame(name = c("Alice", "Bob", "Charlie"),
                     age = c(35, 30, 56),
                     salary = c(50000, 60000, 70000))
    df
    # Sort the data frame by the age variable in ascending order
    df_sorted <- df[order(df$age),]
    df_sorted
    Output:
name age salary
1   Alice  35  50000
2     Bob  30  60000
3 Charlie  56  70000
    After sorting:
    name age salary
2     Bob  30  60000
1   Alice  35  50000
3 Charlie  56  70000
    In the above example, we created a data frame with three variables name, age, and salary, and then sorted the data frame by the age variable in ascending order using the order() function. The resulting data frame df_sorted will have the same variables and observations as the original data frame df, but with the rows rearranged based on the sorted age variable.

  4. Adding a column to a data frame: Use the $ or [[ ]] operator to add a new column to a data frame. For example, to add a new column income to a data frame data based on the salary variable, you can use data$income <- data$salary * 40.

  5. Removing a column from a data frame: Use the $ or [[ ]] operator to remove a column from a data frame. For example, to remove the gender variable from a data frame data, you can use data$gender <- NULL or data[["gender"]] <- NULL.

  6. Renaming a column in a data frame: Use the names() function to rename a column in a data frame. For example, to rename the gender variable to sex in a data frame data, you can use names(data)[names(data) == "gender"] <- "sex".

Extracting,combining and mergeing reshaping data frames

tidyr is a popular R package for data wrangling and manipulation. It provides several functions for transforming data between wide and long formats, as well as for handling missing values and other data cleaning tasks.

Here are some of the commonly used tidyr functions with examples:

  1. gather() - This function is used to convert wide data to long format by gathering columns into key-value pairs.
    Example: Suppose you have a dataset with columns for year, month, and sales for different product categories.
    library(tidyr)
    sales_wide <- data.frame(year = c(2020, 2020, 2021, 2021), month = c("Jan", "Feb", "Jan", "Feb"), "Electronics" = c(1000, 1200, 1100, 1400), "Clothing" = c(800, 900, 950, 1100))
    sales_wide
    Output:
      year month Electronics Clothing
1 2020   Jan        1000      800
2 2020   Feb        1200      900
3 2021   Jan        1100      950
4 2021   Feb        1400     1100
    You can use gather() to convert this wide data to long format, where each row represents a unique observation (year, month, category, and sales).
    sales_long <- gather(sales_wide, key = "category", value = "sales", -c(year, month))
    sales_long
    Output:
        year month    category sales
1 2020   Jan Electronics  1000
2 2020   Feb Electronics  1200
3 2021   Jan Electronics  1100
4 2021   Feb Electronics  1400
5 2020   Jan    Clothing   800
6 2020   Feb    Clothing   900
7 2021   Jan    Clothing   950
8 2021   Feb    Clothing  1100
  2. spread() - This function is used to convert long data to wide format by spreading a key-value column into multiple columns.
    Example: Suppose you have a dataset with columns for year, month, category, and sales.

    library(tidyr)
    sales_long <- data.frame(year = c(2020, 2020, 2021, 2021), month = c("Jan", "Feb", "Jan", "Feb"), category = rep(c("Electronics", "Clothing"), 4), sales = c(1000, 1200, 1100, 1400, 800, 900, 950, 1100))
    sales_long

    Output:
      year month    category sales
1 2020   Jan Electronics  1000
2 2020   Feb    Clothing  1200
3 2021   Jan Electronics  1100
4 2021   Feb    Clothing  1400
5 2020   Jan Electronics   800
6 2020   Feb    Clothing   900
7 2021   Jan Electronics   950
8 2021   Feb    Clothing  1100
    You can use spread() to convert this long data to wide format, where each row represents a unique combination of year, month, and category, and each column represents sales.

    sales_wide <- spread(sales_long, key = "category", value = "month")
    sales_wide

3.separate() - This function is used to separate a single column into multiple columns based on a delimiter.

  3. unite() - This function is used to combine multiple columns into a single column with a delimiter.
    Example: Suppose you have a dataset with columns for year, month, and day.

    library(tidyr)
    dates <- data.frame(year = c(2020, 2020, 2021, 2021), month = c("Jan", "Feb", "Jan", "Feb"), day = c(1, 2, 1, 2))

1 2020   Jan   1
2 2020   Feb   2
3 2021   Jan   1
