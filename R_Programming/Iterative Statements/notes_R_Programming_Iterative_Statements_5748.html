Iterative Statements
​​​​​​​In R, iterative statements allow you to repeat a block of code multiple times. There are several types of iterative statements in R, including for loops, while loops, and the repeat loop.
for loop

It is a type of loop or sequence of statements executed repeatedly until exit condition is reached.

Syntax:

for (variable in sequence) {
    # code to be executed for each value in the sequence
}

Example:

x <- letters[4:10]
for(i in x){
  print(i)
}

Output:
[1] "d"
[1] "e"
[1] "f"
[1] "g"
[1] "h"
[1] "i"
[1] "j"


​​​​​​​

​​​​​​​

while loop

While loop is another kind of loop iterated until a condition is satisfied. The testing expression is checked first before executing the body of loop.The while loop is used to repeat a set of statements as long as a condition is TRUE. ​​​​​​​

Syntax:

while(expression){
    statement
    ....
    ....
}

Example:

x = 1
# Print 1 to 5
while(x <= 5){
  print(x)
  x = x + 1
}

Output:
[1] 1
[1] 2
[1] 3
[1] 4
[1] 5

repeat

repeat loop and break statement
The repeat loop is used to repeat a set of statements an indefinite number of times. The loop can be stopped using the break statement.

Syntax:

repeat {
   statements
   ....
   ....
   if(expression) {
      break
   }
}


Example:

x = 1
# Print 1 to 5
repeat{
  print(x)
  x = x + 1
  if(x > 5){
    break
   }
​​​​​​​}

Output:
[1] 1
[1] 2
[1] 3
[1] 4
[1] 5

nested loops

Nested loops are similar to simple loops. Nested means loops inside loop. Moreover, nested loops are used to manipulate the matrix.

Example:

# Defining matrix
m <- matrix(2:15, 2)
 
for (r in seq(nrow(m))) {
  for (c in seq(ncol(m))) {
    print(m[r, c])
  }
}

Loop Control Statements

Loop control statements are used to control the flow of execution in iterative statements such as for loops and while loops. The two main loop control statements in R are break and next.

  1. The break statement is used to exit a loop early. When a break statement is encountered within a loop, the loop is terminated immediately, and the program continues with the next statement after the loop.

    For example:

    for (i in 1:10) {
        if (i >= 5) {
            break
        }
        print(i)
    }
    ​​​​​​​In this example, the for loop is supposed to iterate 10 times, but the break statement causes the loop to exit early when i >= 5. So, only the numbers 1 to 4 are printed.

  2. The next statement is used to skip the current iteration of a loop and move on to the next iteration. When a next statement is encountered within a loop, the current iteration is skipped, and the program continues with the next iteration of the loop.
    For example:

    for (i in 1:10) {
        if (i %% 2 == 0) {
            next
        }
        print(i)
    }
    Output:
    [1] 1
    [1] 3
    [1] 5
    [1] 7
    [1] 9
    In this example, the next statement causes the program to skip all even numbers, so only the odd numbers from 1 to 10 are printed.
