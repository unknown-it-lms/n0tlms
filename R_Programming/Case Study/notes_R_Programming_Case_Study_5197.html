Case Study
​​​​​​​Examine the built in ChickWeight data (the help gives background about the data). The
function split will prove useful to do the following (as will a script).
  1. Construct a plot of weight against time for chick number 34.
  2. For chicks in diet group 4, display box plots for each time point.
  3. Compute the mean weight for chicks in group 4, for each time point. Plot this mean value against time.
  4. Repeat the previous computation for group 2. Add the mean for group 2 to the existing plot.
  5. Add a legend and a title.
  6. Copy and paste the graph into Word


    # Load the ChickWeight dataset
    data(ChickWeight)

    # Check the structure and content of the dataset
    str(ChickWeight)

    # Plot weight against time for chick number 34
    chick_34 <- subset(ChickWeight, Chick == 34)
    plot(chick_34$Time, chick_34$weight, type = "o", col = "blue",
         xlab = "Time", ylab = "Weight", main = "Weight against Time for Chick 34")

    # Subset data for chicks in diet group 4
    group_4 <- subset(ChickWeight, Diet == 4)

    # Display boxplots for each time point for diet group 4
    boxplot(weight ~ Time, data = group_4, col = "orange",
            xlab = "Time", ylab = "Weight",
            main = "Boxplots of Weight for Diet Group 4")

    # Compute the mean weight for chicks in group 4, for each time point
    mean_weight_group_4 <- aggregate(weight ~ Time, data = group_4, FUN = mean)

    # Plot mean weight against time for group 4
    points(mean_weight_group_4$Time, mean_weight_group_4$weight, type = "o", col = "red")

    # Repeat the computation for group 2
    group_2 <- subset(ChickWeight, Diet == 2)
    mean_weight_group_2 <- aggregate(weight ~ Time, data = group_2, FUN = mean)

    # Add mean for group 2 to the existing plot
    points(mean_weight_group_2$Time, mean_weight_group_2$weight, type = "o", col = "green")

    # Add legend and title
    legend("topright", legend = c("Group 4 Mean", "Group 2 Mean"),
           col = c("red", "green"), pch = 1)
    title(main = "Mean Weight for Diet Groups 4 and 2 over Time")
