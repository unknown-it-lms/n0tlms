Functions

Functions in R are a key part of the language and are used to carry out specific tasks or operations. A function in R is a collection of code that is designed to perform a specific task, such as calculating a mathematical operation or reading data from a file.

​

R has many built-in functions that can be used for common tasks, such as mean(), sum(), and print().
However, you can also create your own functions in R using the following syntax:

function_name <- function(arg1, arg2, ...) {
  # Code to be executed
  return(output)
}

In this example, function_name is the name of the function, and arg1, arg2, and so on are the arguments passed to the function. The code to be executed is placed within the curly braces, and the return statement is used to return the output of the function.

Example of a function that calculates the sum of two numbers:

sum_numbers <- function(a, b) { return(a + b) }

You can then call the function by passing it arguments:

sum_numbers(2, 3)

Output:
[1] 5

R also has many packages that provide additional functions for specialized tasks, such as data manipulation and statistical analysis. You can load these packages using the library() function and then use their functions in your code.

Components of function


​

A typical R function is composed of several components:

  1. Function name: The name of the function, which is used to call the function in R.

  2. Arguments: The input parameters that the function takes. These are enclosed in parentheses after the function name. Arguments can have default values, which are used if the argument is not specified when the function is called.

  3. Body: The code that is executed when the function is called. This can be a single expression or a series of statements enclosed in braces.

  4. Return value: The value that the function returns when it completes its execution. This is specified using the return keyword, followed by the value to be returned.
    return statement is used to return the result of an executed function and returns control to the calling function.
    Syntax:

    return(expression)

  5. Documentation: A description of the function and its usage. This is typically included as a comment block immediately after the function definition, using the #' notation.


Example:

# Checks value is either positive, negative or zero
func <- function(x){
  if(x > 0){
    return("Positive")
  }else if(x < 0){
    return("Negative")
  }else{
    return("Zero")
  }
}

 Function call :

func(1)
func(0)
func(-1)

Output:
[1] "Positive"
[1] "Zero"
[1] "Negative"

Built in functions in r

R comes with a large number of built-in functions that cover a wide range of tasks. Here are some of the most commonly used built-in functions in R:

  1. print(): This function is used to print objects to the console.

    x <- 5
    print(x)
    # Output: [1] 5

  2. length(): This function returns the number of elements in an object.

    x <- c(1, 2, 3, 4, 5)
    length(x)
    # Output: [1] 5

  3. sum(): This function returns the sum of the elements in a vector.

    x <- c(1, 2, 3, 4, 5)
    sum(x)
    # Output: [1] 15

  4. mean(): This function returns the mean of the elements in a vector.

    x <- c(1, 2, 3, 4, 5)
    mean(x)
    # Output: [1] 3

  5. sd(): This function returns the standard deviation of the elements in a vector.

    x <- c(1, 2, 3, 4, 5)
    sd(x)
    # Output: [1] 1.581139

  6. var(): This function returns the variance of the elements in a vector.

    x <- c(1, 2, 3, 4, 5)
    var(x)
    # Output: [1] 2.5

  7. max(): This function returns the maximum value in a vector.

    x <- c(1, 2, 3, 4, 5)
    max(x)
    # Output: [1] 5

  8. min(): This function returns the minimum value in a vector.

    x <- c(1, 2, 3, 4, 5)
    min(x)
    # Output: [1] 1

  9. seq(): This function is used to generate a sequence of numbers.

    seq(from = 1, to = 5, by = 1)
    # Output: [1] 1 2 3 4 5

  10. rep(): This function is used to repeat a value or a sequence of values.

    rep(1, times = 5)
    # Output: [1] 1 1 1 1 1

  11. paste(): This function is used to concatenate strings.

    paste("Hello", "world!")
    # Output: [1] "Hello world!"

  12. substr(): This function is used to extract substrings from a string.
    x <- "Hello world!"
    substr(x, start = 7, stop = 11)
    # Output: [1] "world"

  13. grep(): This function is used to search for a pattern in a string or a vector.

    x <- c("apple", "banana", "orange", "grape")
    grep("a", x)
    # Output: [1] 1 2 3

  14. sort(): This function is used to sort the elements in a vector.

    x <- c(3, 1, 4, 2, 5)
    sort(x)
    # Output: [1] 1 2 3 4 5

  15. order(): This function is used to return the indices that would sort a vector.

    x <- c(3, 1, 4, 2, 5

These are just a few examples of the many built-in functions that R provides. There are many more functions that can be used for data manipulation, statistical analysis, plotting, and more.

Character functions

R provides a variety of character functions that can be used for working with strings and text data. Here are some examples of character functions in R:

  1. nchar(): This function returns the number of characters in a string.
    mystring <- "Hello, World!"
    nchar(mystring)
    Output:
    [1] 13

  2. toupper(): This function converts a string to uppercase.
    mystring <- "hello, world!"
    toupper(mystring)
    Output:
    [1] "HELLO, WORLD!"

  3. tolower(): This function converts a string to lowercase.
    mystring <- "Hello, World!"
    tolower(mystring)
    Output:
    [1] "hello, world!"
    ​​​​​​​
  4. substr(): This function returns a substring of a string.
    mystring <- "Hello, World!"
    substr(mystring, 1, 5)
    Output: [1] "Hello"

  5. paste(): This function concatenates strings together.
    firstname <- "John" lastname <- "Doe"
    paste(firstname, lastname)
    Output:
    [1] "John Doe"

  6. gsub(): This function replaces all occurrences of a substring in a string.
    mystring <- "Hello, World!"
    gsub("l", "z", mystring)
    Output:
    [1] "Hezzo, Worzd!"
  1. strsplit(): This function splits a string into a list of substrings based on a delimiter.
    mystring <- "apple,banana,orange"
    strsplit(mystring, ",")
    Output:
    [[1]] # [1] "apple" "banana" "orange"
  1. paste0(): This function concatenates strings together without any separator.
    firstname <- "John"
    lastname <- "Doe"
    paste0(firstname, lastname)
    Output:
    ​​​​​​​[1] "JohnDoe"
These are just a few examples of the many character functions that R provides. There are many more functions that can be used for manipulating and processing strings in R.
