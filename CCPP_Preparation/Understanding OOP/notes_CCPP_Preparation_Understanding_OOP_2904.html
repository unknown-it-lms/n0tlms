1What are principles of Object oriented programming?    

Major Principles: Abstraction, Encapsulation, Inheritance and Polymorohims
Minor Principles: Persistance, Concurrency and strong typing
2What are advantages of Object Oriented Programming?

Classes are the basic building blocks of OOP code. They model the real life entities quite well. They also model the relationships between real life entities very well though the concepts of Inheritance, association. Hence OOP based software design is closer to real life.
Access to data is restricted only to selected functions hence it is more robust. 
Because of inheriance, the code is easy to reuse and extend.
3What is class and object? Give real-life example.

Classes are created to model the real life entities in a software system. 
Objects are instances created from these classes. 
For example, In a banking software, class BankAccount models concept of bank account in real bank. Various instances of this class BankAccount are created to model bank accounts of various account holders in a bank.
Other examples:
class Student models the student entity > Objects of class student created for each student.
class Payment models the payment entity > Instances of this class Payment are created for each transaction that takes place actually.
