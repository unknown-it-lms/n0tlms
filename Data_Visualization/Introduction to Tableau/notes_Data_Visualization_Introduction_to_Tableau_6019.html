Introduction to Tableau
​​​​​​​Tableau is a powerful data visualization and business intelligence tool that helps people analyze, understand, and present data in a visually appealing and interactive way. It was developed by Tableau Software, which was acquired by Salesforce in 2019.
​

Tableau provides a user-friendly interface that allows users to connect to various data sources, such as spreadsheets, databases, and cloud services, and create interactive visualizations, reports, and dashboards. It supports a wide range of data types, including text, numbers, dates, and geographical information.

One of the key strengths of Tableau is its ability to handle large and complex datasets, allowing users to explore and gain insights from vast amounts of information. It employs a drag-and-drop approach, which makes it easy for users to create visualizations and perform data analysis without the need for complex coding or scripting.

Tableau offers a variety of visualization options, including bar charts, line graphs, scatter plots, maps, and more. Users can customize the appearance and behavior of visualizations to highlight specific trends, patterns, or outliers in the data. Additionally, Tableau provides features like filters, parameters, and calculated fields to further refine and manipulate data.

Another notable feature of Tableau is its interactivity. Users can create interactive dashboards that allow viewers to explore data by interacting with the visualizations. This enables data-driven decision-making by enabling users to drill down into specific details, filter data based on various criteria, and view different perspectives of the data.

Tableau also offers options for collaboration and sharing. Users can publish their visualizations and dashboards to Tableau Server or Tableau Public, which allows others to access and interact with the visualizations through a web browser. This facilitates data sharing, collaboration, and enables stakeholders to stay informed and make informed decisions based on the latest data.

In summary, Tableau is a widely used data visualization and business intelligence tool that empowers users to explore, analyze, and present data in a visual and interactive manner. It helps businesses and organizations make data-driven decisions by uncovering insights and patterns that may not be apparent in raw data alone.

Different Tableau Products:
Tableau offers several products that cater to different data visualization and business intelligence needs.

1. Tableau Desktop:
Tableau Desktop is the main data visualization and analysis tool. It allows users to connect to various data sources, create interactive visualizations, build dashboards, and perform data analysis. It is a powerful tool for data exploration, ad-hoc analysis, and creating rich and interactive visualizations.

2. Tableau Server:
Tableau Server enables users to publish, share, and collaborate on Tableau visualizations within an organization. It provides a web-based platform where users can access and interact with visualizations, create dashboards, and schedule data refreshes. Tableau Server also offers security features to control access to data and visualizations.

3. Tableau Online:
Tableau Online is a cloud-based version of Tableau Server. It allows users to publish and share Tableau visualizations on the web without the need for infrastructure setup and maintenance. With Tableau Online, users can collaborate, access dashboards from anywhere, and enjoy the benefits of cloud-based data storage and scalability.

4. Tableau Prep:
Tableau Prep is a data preparation tool that helps users clean, shape, and transform their data for analysis. It offers a visual interface for performing data cleansing, combining multiple data sources, splitting columns, creating calculations, and more. Tableau Prep allows users to streamline and automate data preparation tasks before visualizing the data in Tableau Desktop.

5. Tableau Mobile:
Tableau Mobile is a companion app that allows users to access and interact with Tableau visualizations on mobile devices, such as smartphones and tablets. It provides a responsive and touch-friendly interface, enabling users to explore and share data insights while on the go.

6. Tableau Public:
Tableau Public is a free version of Tableau that allows users to create and publish visualizations to the web. Users can share their interactive dashboards and visualizations publicly, making them accessible to anyone. Tableau Public is often used by journalists, bloggers, and data enthusiasts to create and share data stories.​​​​​​​

Modern Data Analytics Tools
​​​​​​​In addition to Tableau, there are several other modern data analytics tools available in the market.

1. PowerBI:
Power BI is a business analytics tool by Microsoft. It allows users to connect to various data sources, create interactive dashboards, and perform data analysis. Power BI offers a user-friendly interface, robust data visualization capabilities, and integration with other Microsoft products like Excel and Azure.

​
2. QlikView and Qlik Sense:
QlikView and Qlik Sense are data discovery and visualization tools offered by Qlik. They enable users to explore and analyze data, create interactive dashboards, and generate data-driven insights. Qlik products emphasize associative data indexing, allowing users to explore data relationships dynamically.

​
3. Looker:
Looker is a cloud-based data platform that provides data exploration, visualization, and business intelligence capabilities. It offers a modeling language called LookML, which helps define data models and relationships. Looker focuses on providing a unified view of data across an organization and enables collaboration through its shared data experiences.

​
4. Domo:
Domo is a cloud-based platform that combines data visualization, analytics, and collaboration tools. It allows users to connect to various data sources, create dashboards, and share insights with others. Domo also offers features like data integration, data preparation, and automated alerts and notifications.

​
5. Google Data Studio:
Google Data Studio is a free data visualization and reporting tool offered by Google. It allows users to create interactive dashboards and reports using data from various sources, including Google Analytics, Google Sheets, and databases. Google Data Studio emphasizes collaboration and sharing capabilities.

​
6.  SAS Visual Analytics:
SAS Visual Analytics is a comprehensive data visualization and analytics tool provided by SAS. It enables users to explore and analyze data visually, create interactive reports, and perform advanced analytics. SAS Visual Analytics offers integration with other SAS products for advanced statistical analysis and modeling.

​
7. Apache Superset:
Apache Superset is an open-source data exploration and visualization platform. It provides a user-friendly interface for creating interactive dashboards, exploring data, and sharing insights. Apache Superset supports various data sources and offers extensibility through custom visualization plugins.​​​​​​​

Data Sources in Tableau
​​​​​​​Tableau can connect to a wide range of data sources, enabling users to access and visualize data from various systems.

​

1. Relational Databases:
Tableau supports connecting to popular relational databases such as Microsoft SQL Server, Oracle, MySQL, PostgreSQL, and IBM DB2. Users can extract data from these databases directly or establish live connections for real-time data analysis.

​

2. Cloud Databases:
Tableau can connect to cloud-based databases like Amazon Redshift, Google BigQuery, and Microsoft Azure SQL Database. These databases offer scalability and flexibility for handling large volumes of data in the cloud environment.

​

3. Spreadsheets and Flat Files:
Tableau can read data from spreadsheets (e.g., Microsoft Excel, Google Sheets) and flat files (e.g., CSV, TXT). Users can import data from these files or establish a connection to them to automatically update visualizations when the data changes.

4. Web Data Connector:
Tableau provides web data connectors that allow users to connect to data from web services and APIs. With these connectors, users can access data from sources like Salesforce, Google Analytics, Adobe Analytics, and more.

5. Cloud Storage Platform:
Tableau supports integration with cloud storage platforms like Amazon S3, Microsoft Azure Blob Storage, and Google Cloud Storage. Users can connect to data stored in these platforms and analyze it directly in Tableau.

6. Data Wareshouses:
Tableau can connect to data warehouses, including Amazon Redshift, Snowflake, and Google BigQuery. These data warehousing solutions offer powerful data storage and querying capabilities, making it easier to handle large datasets for analysis.

7. Hadoop and NoSQL Databases:
Tableau has built-in connectors for Hadoop-based systems like Apache Hadoop and Cloudera, as well as NoSQL databases like MongoDB and Cassandra. This allows users to work with data stored in these distributed and non-relational databases.

8. Cloud Applications:
Tableau integrates with cloud applications like Salesforce, Marketo, and ServiceNow. This enables users to connect to these applications and analyze data directly within Tableau.

9. OData:
Tableau supports Open Data Protocol (OData), which allows users to connect to data sources that expose OData endpoints. This includes sources like Microsoft SharePoint and SAP.​​​
