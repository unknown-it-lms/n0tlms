MS EXCEL
​​​​​​​Microsoft Excel is a powerful spreadsheet application developed by Microsoft. It is widely used for data analysis, calculation, visualization, and storage. Excel provides a range of features and functionalities that make it a versatile tool for various tasks, including financial analysis, budgeting, project management, and data manipulation.
Microsoft Excel Documentation: Excel help & learning

Some key features and capabilities of MS Excel:

  1. Spreadsheet functionality: Excel is primarily known for its ability to create and manage spreadsheets. It provides a grid-based interface where users can enter and organize data into cells, rows, and columns. Users can perform calculations, apply formulas and functions, and format data to suit their needs.

  2. Formulas and Functions: Excel offers a vast library of built-in formulas and functions that allow users to perform complex calculations and data manipulations. These include arithmetic operations, statistical functions, logical operations, date and time functions, lookup and reference functions, and more. Users can also create custom formulas using Excel's formula language.

  3. Data Analysis: Excel provides various tools for data analysis, such as sorting, filtering, and conditional formatting. It allows users to perform data validation, create data tables, and generate charts and graphs for data visualization. Excel also supports pivot tables, which enable users to summarize and analyze large datasets efficiently.

  4. Data Import and Export: Excel supports importing data from various external sources, including databases, text files, CSV files, and online sources. It provides features for data cleansing and transformation during the import process. Additionally, Excel allows users to export data to different file formats, including PDF, CSV, and HTML.

  5. Collaboration and Sharing: Excel facilitates collaboration and sharing of spreadsheets among multiple users. It supports features like track changes, comments, and real-time co-authoring, allowing users to work simultaneously on the same spreadsheet. Excel also integrates with Microsoft SharePoint and OneDrive, enabling easy sharing and access to spreadsheets online.

  6. Automation: Excel provides automation capabilities through Visual Basic for Applications (VBA), a programming language integrated into Excel. Users can create macros and automate repetitive tasks, customize Excel's functionality, and build interactive user interfaces using VBA.

  7. Add-ins and Extensions: Excel supports add-ins and extensions that extend its functionality. These add-ins can provide additional tools, features, and integration with other software applications. Excel also integrates with other Microsoft Office applications, such as Word and PowerPoint, allowing for seamless data sharing and integration.

These are just a few highlights of what Excel offers. With its extensive range of features, Excel is widely used in various industries and professions, including finance, accounting, data analysis, project management, and research, among others.

Some basics of Microsoft Excel:

  1. Workbook: Excel files are called workbooks. A workbook can contain multiple worksheets. By default, a new workbook opens with three worksheets.

    ​

  2. Worksheet: A worksheet is a grid of cells arranged in rows and columns. It is where you enter and manipulate data. Each worksheet has a name displayed at the bottom of the Excel window.

    ​

  3. Cells: Cells are the individual rectangular boxes within a worksheet. They are identified by a combination of column letters and row numbers. For example, cell A1 refers to the cell in the first column and the first row.

    ​

  4. Data Entry: You can enter text, numbers, or formulas into cells. Simply select a cell, type the content, and press Enter to move to the next cell. To edit a cell, double-click on it or press F2.

  5. Formulas: Formulas are used to perform calculations in Excel. They begin with an equal (=) sign. For example, the formula =A1+B1 adds the values in cells A1 and B1. Excel provides a wide range of functions for various calculations, such as SUM, AVERAGE, COUNT, MAX, MIN, and more.

  6. Autofill: Autofill is a convenient feature that allows you to quickly fill a series of cells with a pattern or sequence. You can click and drag the fill handle (a small square at the bottom-right corner of a selected cell) to fill adjacent cells with values, dates, or formulas.

  7. Formatting: Excel provides various formatting options to change the appearance of cells, such as font style, size, color, and alignment. You can format numbers as currency, percentages, dates, and more. Conditional formatting allows you to highlight cells based on specific criteria.

  8. Charts and Graphs: Excel enables you to create visual representations of your data using charts and graphs. Select the data range, click on the "Insert" tab, and choose the desired chart type (e.g., column chart, line chart, pie chart). Excel provides customization options to modify the chart's appearance and labels.

  9. Sorting and Filtering: You can sort data in Excel based on column values in ascending or descending order. Filtering allows you to display only specific data based on criteria you define. These features help you organize and analyze data more effectively.

  10. Saving and Printing: To save your workbook, click on the "File" tab and choose "Save" or press Ctrl + S. You can save in various formats, including .xlsx, .csv, and .pdf. When you need a hard copy, you can print your worksheets using the "File" tab and selecting "Print."

These basics will help you get started with Microsoft Excel. Excel offers many more advanced features and functionalities, but understanding these fundamentals is a great foundation for working with spreadsheets and analyzing data.

Functions
​​​​​​​Excel provides a vast array of built-in functions that allow you to perform calculations, manipulate data, and analyze information.
Microsoft Excel Documentation: Formulas and functions
Some commonly used categories of functions in Excel:
  1. Mathematical Functions:
  • SUM: Adds up a range of numbers. Syntax: =SUM(number1, number2, ...)

  • AVERAGE: Calculates the average of a range of numbers. Syntax: =AVERAGE(number1, number2, ...)

  • MAX: Returns the highest value in a range. Syntax: =MAX(number1, number2, ...)

  • MIN: Returns the lowest value in a range. Syntax: =MIN(number1, number2, ...)

  • ROUND: Rounds a number to a specified number of decimal places.
    Syntax: =ROUND(number, num_digits)

  • SQRT: Calculates the square root of a number. Syntax: =SQRT(number)

  1. Text Functions:

  • CONCATENATE: Joins multiple text strings together. Syntax: =CONCATENATE(text1, text2, ...)

  • LEFT: Extracts a specified number of characters from the left side of a text string.
    Syntax: =LEFT(text, num_chars)

  • RIGHT: Extracts a specified number of characters from the right side of a text string.
    Syntax: =RIGHT(text, num_chars)

  • LEN: Calculates the length (number of characters) of a text string. Syntax: =LEN(text)

  • SUBSTITUTE: Replaces specific text within a text string.
    Syntax: =SUBSTITUTE(text, old_text, new_text, [instance_num])

  1. Date and Time Functions:

  • TODAY: Returns the current date. Syntax: =TODAY()

  • NOW: Returns the current date and time. Syntax: =NOW()

  • DATE: Creates a date from specified year, month, and day values. Syntax: =DATE(year, month, day)

  • YEAR: Extracts the year from a given date. Syntax: =YEAR(date)

  • MONTH: Extracts the month from a given date. Syntax: =MONTH(date)

  • DAY: Extracts the day from a given date. Syntax: =DAY(date)

  1. Logical Functions:

  • IF: Performs a specified action based on a condition.
    Syntax: =IF(logical_test, value_if_true, value_if_false)

  • AND: Returns TRUE if all conditions are TRUE.
    Syntax: =AND(condition1, condition2, ...)

  • OR: Returns TRUE if at least one condition is TRUE. Syntax: =OR(condition1, condition2, ...)

  • NOT: Reverses the logical value of a condition. Syntax: =NOT(logical)

  1. Lookup and Reference Functions:

  • VLOOKUP: Searches for a value in the first column of a range and returns a corresponding value from another column. Syntax: =VLOOKUP(lookup_value, table_array, col_index_num, [range_lookup])

  • HLOOKUP: Searches for a value in the first row of a range and returns a corresponding value from another row. Syntax: =HLOOKUP(lookup_value, table_array, row_index_num, [range_lookup])

  • INDEX: Returns the value of a cell in a specified row and column.
    Syntax: =INDEX(array, row_num, [column_num])

  • MATCH: Searches for a specified value in a range and returns its relative position.
    Syntax: =MATCH(lookup_value, lookup_array, [match_type])

  1. Statistical Functions:

  • COUNT: Counts the number of cells in a range that contains numbers.
    Syntax: =COUNT(value1, value2, ...)

  • COUNTA: Counts the number of cells in a range that is not empty.
    Syntax: =COUNTA(value1, value2, ...)

  • AVERAGEIF: Calculates the average of a range based on a specified condition.
    Syntax: =AVERAGEIF(range, criteria, [average_range])

  • STDEV: Estimates the standard deviation based on a sample. Syntax: =STDEV(number1, number2, ...)

  • CORREL: Calculates the correlation coefficient between two data sets. Syntax: =CORREL(array1, array2)

These are just a few examples of the numerous functions available in Excel. You can access and explore the complete list of functions by clicking on the "Formulas" tab in Excel and choosing the relevant function category.

Formula
​​​​​​​Formulas in Excel are used to perform calculations, manipulate data, and derive meaningful results. Excel provides a wide range of built-in functions and operators that can be combined within formulas to achieve various tasks.
Microsoft Excel Documentation: Formulas and functions
Some key aspects of using formulas in Excel:

  1. Syntax: Excel formulas begin with an equal (=) sign, followed by the function name or operator and the arguments enclosed in parentheses. For example, =SUM(A1:A5) calculates the sum of the values in cells A1 to A5.

  2. Cell References: Cell references allow you to refer to specific cells or ranges within a formula. They help dynamically update the formula when the referenced cells' values change. For example, =A1+B1 adds the values of cells A1 and B1.

  3. Functions: Excel provides a vast library of built-in functions that perform specific calculations or tasks. Functions are used to carry out various operations such as mathematical calculations, date and time calculations, text manipulation, statistical analysis, and more. For example, =AVERAGE(A1:A5) calculates the average of the values in cells A1 to A5.

  4. Operators: Excel supports various operators to perform mathematical and logical operations within formulas. Commonly used operators include addition (+), subtraction (-), multiplication (*), division (/), exponentiation (^), comparison operators (>, <, =), and logical operators (AND, OR, NOT). For example, =A1+B1 uses the addition operator to sum the values in cells A1 and B1.

  5. Parentheses: Parentheses are used to group elements within a formula to control the order of calculations and prioritize certain operations. For example, =(A1+B1)*C1 calculates the sum of A1 and B1 first, and then multiplies the result by the value in C1.

  6. AutoFill: Excel's AutoFill feature allows you to quickly copy and extend formulas to adjacent cells in a series or pattern. Simply click and drag the fill handle (a small square at the bottom-right corner of a selected cell) to populate the desired range with the formula.

  7. Error Handling: When a formula encounters an error, such as dividing by zero or referencing an invalid cell, Excel displays an error value. You can use error-handling functions like IFERROR, ISERROR, and ISNA to handle or suppress these errors in your formulas.

Excel's formula capabilities are extensive, allowing you to perform complex calculations, data transformations, and analysis tasks. By leveraging formulas effectively, you can automate calculations, create dynamic reports, and extract valuable insights from your data.

Charts
​​​​​​​Microsoft Excel provides powerful charting capabilities that allow you to visually represent and analyze data.
An overview of creating charts in Excel:

  1. Select Data: Before creating a chart, you need to select the data you want to include in the chart. The data can be in adjacent columns or rows within a worksheet or located in different worksheets or workbooks.

    ​

  2. Insert Chart: Once the data is selected, go to the "Insert" tab in the Excel ribbon and click on the desired chart type from the "Charts" group. Excel offers a variety of chart types, including column, bar, line, pie, scatter, and more.

    ​

  3. Customize Chart: After inserting a chart, you can customize it to suit your needs. Right-click on different elements of the chart (such as the chart title, axis labels, or data series) to access various formatting options. You can change colors, fonts, sizes, gridlines, legends, and other chart elements.

    ​

  4. Chart Design and Format: The "Chart Design" tab provides additional options to refine and enhance the chart's appearance. It allows you to modify chart layouts, switch row and column data, add data labels, adjust axis scales, and apply predefined chart styles.

    ​

  5. Data Labels and Legends: Data labels provide additional information about individual data points in the chart, such as their values or percentage. You can add data labels to the chart elements by right-clicking on them and selecting the appropriate option. Legends help identify different series or categories in the chart, and you can customize their position and appearance.

    ​

  6. Chart Filters: Excel allows you to filter data directly within the chart by using chart filters. Clicking on the filter icon in the top-right corner of the chart allows you to selectively display or hide specific series, categories, or data points.

    ​

  7. Chart Titles and Labels: To make your chart more informative, you can add titles and labels. Chart titles help describe the purpose or context of the chart, while axis titles provide labels for the horizontal (X) and vertical (Y) axes. You can add or modify titles and labels by selecting the corresponding chart element and editing the text.

    ​

  8. Chart Types and Switching: If you want to change the chart type or experiment with different visualizations, you can do so using the "Change Chart Type" option. It lets you switch between various chart types while preserving the underlying data.

    ​

  9. Updating Data: After creating a chart, if you update or modify the underlying data, Excel provides an option to automatically update the chart. You can right-click on the chart and choose "Edit Data" to make changes directly within the Excel spreadsheet.

    ​

By effectively utilizing Excel's charting capabilities, you can create professional and insightful visuals to analyze and present your data in a visually appealing manner.

Microsoft Excel offers a wide range of chart types to visually represent data.

Microsoft Excel Documentation: Available chart types in Office

Some commonly used chart types in Excel:

  1. Column Chart: Displays data in vertical columns. It is suitable for comparing values across different categories or showing changes over time.

    ​

  2. Bar Chart: Similar to a column chart, but the bars are horizontal instead of vertical. It is useful for comparing values between different categories.

    ​

  3. Line Chart: Shows data as a series of data points connected by straight lines. It is effective for displaying trends over time or continuous data.

    ​

  4. Pie Chart: Represents data as slices of a pie, where each slice represents a percentage or proportion of the whole. It is ideal for illustrating the composition of a whole or comparing parts to the whole.

    ​

  5. Scatter Plot: Represents data points as individual dots on a two-dimensional chart. It helps analyze the relationship between two variables and identify patterns or correlations.

    ​

  6. Area Chart: Similar to a line chart, but the area beneath the line is shaded. It is useful for visualizing cumulative data or comparing multiple series.

    ​

  7. Radar Chart: Displays data points on a circular grid with multiple axes. It is suitable for comparing multiple quantitative variables.

    ​

  8. Bubble Chart: Represents data points as bubbles, where the size of each bubble represents a third variable. It is useful for showing relationships between three variables.

    ​

  9. Stock Chart: Specifically designed for financial data, it shows the high, low, opening, and closing prices of a stock or other financial instruments over a specific period.

    ​

  10. Surface Chart: Presents data in a three-dimensional format with x, y, and z axes. It is suitable for showing relationships between three variables and their values across a continuous surface.

    ​

  11. Combo Chart: Allows combining different chart types in a single chart. For example, you can combine a column chart and a line chart to represent two different data series.

    ​

  12. Gantt Chart: Displays project schedules and timelines, illustrating tasks, start dates, durations, and dependencies.

    ​

These are just a few examples of the chart types available in Microsoft Excel. Each chart type has its own purpose and can be customized with various formatting options to suit your specific data visualization needs.

Pivots and Lookups
​​​​​​​I) Pivots:
A PivotTable is a powerful data summarization and analysis tool in Excel that allows you to organize and analyze large sets of data quickly and efficiently. It helps you gain insights from your data by summarizing it into a more manageable and meaningful format.
Microsoft Excel Documentation: Create a PivotTable to analyze worksheet data
A step-by-step explanation of creating a PivotTable in Excel:

  1. Select your data: Start by selecting the range of cells or table that contains the data you want to analyze. Make sure your data is well-organized with column headers.

  2. Insert a PivotTable: Go to the "Insert" tab on the Excel ribbon and click on the "PivotTable" button. A dialog box will appear.

    ​

  3. Choose your data range: In the PivotTable dialog box, ensure that the "Select a table or range" option is selected. The range should automatically be populated based on your selection in step 1. If not, you can manually enter the range.

    ​

  4. Choose where to place the PivotTable: Select whether you want to place the PivotTable on a new worksheet or an existing one. This determines where the PivotTable report will be displayed.

  5. Design your PivotTable: After clicking "OK," you will see the PivotTable Field List.

    ​

    This pane allows you to customize your PivotTable by dragging and dropping fields into different areas:

    • Rows: Drag the fields you want to use as rows into this area. Each unique value in these fields will become a separate row in the PivotTable.

    • Columns: Drag the fields you want to use as columns into this area. Each unique value in these fields will become a separate column in the PivotTable.

    • Values: Drag the fields you want to summarize or analyze into this area. You can choose from various summary functions like sum, average, count, etc. Excel will automatically calculate the summary for each combination of rows and columns.

    • Filters: Drag the fields you want to use as filters into this area. Filters allow you to focus on specific subsets of data based on the values you select.

  6. Customize your PivotTable: You can further customize your PivotTable by right-clicking on it and accessing options such as sorting, filtering, grouping, and formatting. These options help you organize and present your data in a way that suits your needs.

    ​

By using a PivotTable, you can quickly analyze and summarize large datasets, spot trends and patterns, compare data from different perspectives, and create interactive reports. PivotTables are dynamic, so if your underlying data changes, you can easily refresh the PivotTable to reflect the updated information.

II) Lookups:
Lookup functions in Excel are designed to help you find specific values within a range of cells and retrieve related information. They are useful when you need to search for data in a large dataset or match values between different tables. Excel offers several lookup functions, including VLOOKUP, HLOOKUP, INDEX-MATCH, and XLOOKUP.
Microsoft Excel Documentation: LOOKUP function

1. VLOOKUP (Vertical Lookup): 
VLOOKUP searches for a value in the leftmost column of a range or table and returns a value in the same row from a specified column.
The syntax of VLOOKUP is as follows: 
                                             =VLOOKUP(lookup_value, table_range, col_index, [range_lookup])

  • lookup_value: The value you want to search for.
  • table_range: The range of cells that contains the lookup table, including the lookup column and the columns from which you want to retrieve the result.
  • col_index: The column number in the table_range from which you want to retrieve the result.
  • range_lookup: An optional argument that specifies whether to perform an exact match (FALSE) or an approximate match (TRUE or omitted for an approximate match). Exact matches require the data to be sorted in ascending order.
​

2. HLOOKUP (Horizontal Lookup):
HLOOKUP is similar to VLOOKUP but searches for a value in the topmost row of a range or table and retrieves a value from the same column in a specified row. The syntax is the same as VLOOKUP but with the rows and columns flipped.



​

3. INDEX-MATCH:
INDEX-MATCH is a powerful combination of two functions that allows you to perform advanced lookup operations. It overcomes some limitations of VLOOKUP and HLOOKUP, such as the requirement for the lookup column to be the leftmost column.
The syntax of INDEX-MATCH is as follows:
                          =INDEX(column_range, MATCH(lookup_value, lookup_range, [match_type]))

column_range: The range of cells that contains the column from which you want to retrieve the result.
lookup_value: The value you want to search for.
lookup_range: The range of cells that contains the values you want to match against.
match_type: An optional argument that specifies the match type: 0 for an exact match, 1 for a greater than or equal to match, and -1 for a less than or equal to match.

INDEX returns a value from a specified position within a range, and MATCH returns the relative position of a value within a range. By combining them, you can perform more flexible and versatile lookups.

Data Analysis Tool pack
​​​​​​​The Data Analysis Toolpack is an add-in in Microsoft Excel that provides advanced data analysis capabilities. It offers a range of statistical and analytical tools to help you analyze data, make predictions, and gain insights. However, please note that the Data Analysis Toolpack is not available by default in Excel and needs to be enabled manually.
Microsoft Excel Documentation: Load the Analysis ToolPak in Excel

  1. Enable the Data Analysis Toolpack:

    ​

    a. Open Excel and click on the "File" tab in the ribbon.
    b. Select "Options" from the menu.
    c. In the Excel Options dialog box, choose "Add-Ins" from the left-hand side.
    d. In the Manage box at the bottom, select "Excel Add-ins" and click on the "Go" button.
    e. In the Add-Ins dialog box, check the box next to "Analysis ToolPak" and click "OK". f. Excel will now enable the Data Analysis Toolpack, and you should see a new "Data Analysis" option in the "Data" tab of the ribbon.

  2. Using the Data Analysis Toolpack:

    ​

    a. Click on the "Data" tab in the Excel ribbon.
    b. In the "Analysis" group, click on the "Data Analysis" button.
    c. A dialog box will appear with a list of available analysis tools. Choose the desired tool based on your analysis needs.
    d. Select the range of data for analysis by either typing the range or using the "Select Range" button. e. Configure the options and parameters specific to the chosen analysis tool. f. Click "OK" to perform the analysis.

Excel produces the following Summary Output (rounded to 3 decimal places).
R Square
R Square equals 0.962, which is a very good fit. 96% of the variation in Quantity Sold is explained by the independent variables Price and Advertising. The closer to 1, the better the regression line (read on) fits the data.


​
Significance F and P-values
To check if your results are reliable (statistically significant), look at Significance F (0.001). If this value is less than 0.05, you're OK. If Significance F is greater than 0.05, it's probably better to stop using this set of independent variables. Delete a variable with a high P-value (greater than 0.05) and rerun the regression until Significance F drops below 0.05.
Most or all P-values should be below below 0.05. In our example this is the case. (0.000, 0.001 and 0.005).



​
Coefficients
The regression line is: y = Quantity Sold = 8536.214 -835.722 * Price + 0.592 * Advertising. In other words, for each unit increase in price, Quantity Sold decreases with 835.722 units. For each unit increase in Advertising, Quantity Sold increases with 0.592 units. This is valuable information.
You can also use these coefficients to do a forecast. For example, if price equals $4 and Advertising equals $3000, you might be able to achieve a Quantity Sold of 8536.214 -835.722 * 4 + 0.592 * 3000 = 6970.

Residuals
The residuals show you how far away the actual data points are fom the predicted data points (using the equation). For example, the first data point equals 8500. Using the equation, the predicted data point equals 8536.214 -835.722 * 2 + 0.592 * 2800 = 8523.009, giving a residual of 8500 - 8523.009 = -23.009.
