Creating simple table
The CREATE TABLE statement allows you to create a new table in a database.

CREATE TABLE  table_name(
                                                    column_1_definition,
                                                    column_2_definition,
                                                    : ,
                                                     table_constraints
                                                 );
In the above syntax,
Table name must be unique within a database.
If table already exists it generates an error while creation.  If this is the case, MySQL will ignore the whole statement and will not create any new table.

INSERT Clause

Selecting Records(Rows) from table
SQL SELECT statement is used to  retrieve data from a table in the database. A query may retrieve information from specified columns or from all of the columns in the table. To write a simple SQL SELECT Statement, we need to specify the column(s) name and the table name.
The whole query is called SQL SELECT Statement.
Syntax of SQL SELECT Statement:
  
SELECT column_list
FROM table-name
[WHERE Clause]
[GROUP BY clause]
[HAVING clause]
[ORDER BY clause];   

In the above syntax,
  • table-name is the name of the table from which the information is retrieved.
  • column_list includes one or more columns from which data is retrieved.
  • The code within the brackets is optional.
