Notes: MySQL- Alter clause

Alter clause
A Databse Administrator can make changes to the table structure or column definitions after the table has been created in the database. The DDL command ALTER TABLE is used to perform such actions. Alter command provides  multiple utilities exclusive  for schema objects.
The ALTER TABLE statement is used to
  •  Rename a Table/Column
  •  Drop a column
  •  Add a new column
  •  Changing datatype of existing column 
Changing table structure
 Rename 
1. Rename table
Rename is used to change the name of the  table.
Below ALTER TABLE statement renames the table EMP to EMP_NEW.
Syntax - 1 
ALTER  TABLE   table_name
RENAME  TO  new_table_name;
e.g
ALTER  TABLE   EMP
RENAME  TO  EMP_NEW;     
OR
Syntax - 2
RENAME  TABLE  oldtablename TO  newtablename;

2. Rename column 
Rename column will rename the existing column.
Below ALTER TABLE statement renames the column TESTCOL to TESTNEW.
Syntax:
ALTER  TABLE  table_name
RENAME  COLUMN  column_name  TO  new_column_name;
e.g
ALTER  TABLE  EMP_NEW
RENAME  COLUMN  TESTCOL  TO  TESTNEW
also
ALTER  TABLE  table_name
CHANGE  old_column_name  new_column_name  data_type ;
Add New Column 
Add will add new column in the existing table. This change is at the structure level and cannot be reverted.
Below statement adds new column TESTCOL to the EMP_NEW table.
Syntax:
ALTER  TABLE table_name
ADD  column_name data_type;
e.g
ALTER  TABLE EMP_NEW
ADD  TESTCOL VARCHAR(100) ;
Changing datatype of existing column
Very rearly though, there can be a need to change the datatype of an existing column. For this purpose,
Modify clause is used.
Syntax:
ALTER TABLE table_name    
MODIFY column_name datatype;     
e.g
Let us change empno from emp table which is int to varchar data.
ALTER  TABLE  emp   
MODIFY  empno varchar(10):
Delete existing column (Drop)
To delete any column from the table, drop column statement is used. This change is at a structure level and cannot be reverted.
Syntax :
 ALTER  TABLE tble_name
 DROP  COLUMN column_name;
e.g
Below ALTER TABLE statement drops the column TESTNEW from EMP_NEW table
               ALTER TABLE EMP_NEW
               DROP COLUMN TESTNEW;
