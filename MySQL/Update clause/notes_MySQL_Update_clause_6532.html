Notes: MySQL- Update clause

Update clause
Sometimes we need to make changes in existing data stored in the tables. The data stored in table is always stored permanently. So to make changes (update or modify) in the data, RDBMS has given us UPDATE Clause. 
Value of a column in the table is modified and it is necessary to specify all column names correctly for whome the value needs to be modified. One need to handle update clause carefully as default scope of Update Clause is entire table. This scope can be restricted using where clause . 
Updating Column
​​​​​Sometimes there is a need to make changes in the data stored in the database. To change existing data from the table, UPDATE statement is used. This will modify current values stored in the database. New values given to the column in update command will now be stored in the table. Old values will be over written.

UPDATE table_name
SET column1 = value1, column2 = value2, ...
[ WHERE condition ];

UPDATE Customers
SET ContactName = 'Alfred Schmidt', City= 'Frankfurt'
WHERE CustomerID = 1;

UPDATE Customers
SET ContactName='Juan';
Using update
Syntax:     

UPDATE  table_name
SET [column_name1= value1,...column_nameN = valueN]
[WHERE condition]   

For example:
        
UPDATE   student  
SET  fname = 'Jhon', sname= 'Wick'
