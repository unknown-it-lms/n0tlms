Notes: MySQL- Triggers

Concept of Trigger 
 Need of triggers
Triggers are special procedures which get called automatically if some action occurs.

Actions on which the triggers are fired
  •    before
  •    after
  •    insteadof ----- this is used with view ---- does not work in mysql

Triggers are called on delete,update,insert DML statements.
 
Triggers work on   

Database level --- works only oracle
   before database shutdown
   after databse start   
   after login
   before logout

level
   statement level ---- not possible in mysql
   row level




create table course_audit
(old_cid int,
new_cid int,
chdt date,
user varchar(20),
action varchar(20),
ofid int,
nfid int);

delimiter //     

create trigger upd_course after update  on course
for each row
begin
  insert into course_audit values(OLD.cid,NEW.cid,curdate(),user(),'update',OLD.fid,NEW.fid);
end//

delimiter ;

delimiter //     

create trigger del_course after delete  on course
for each row
begin
  insert into course_audit values(OLD.cid,null,curdate(),user(),'delete',OLD.fid,null);
end//

delimiter ;

delimiter //
create trigger ins_course before insert  on course
for each row
begin
  insert into course_audit values(null,NEW.cid,curdate(),user(),'insert',null,NEW.fid);
end//

delimiter ;
