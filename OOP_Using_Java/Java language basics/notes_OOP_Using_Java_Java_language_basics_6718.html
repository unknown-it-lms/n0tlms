Conditional statements
Java provides statements that can be used to control the flow of Java code. Such statements are called control flow statements. Java provides three types of control flow statements.
  1. Decision Making statements
    • if statements
    • switch statement
  2. Loop statements
    • do while loop
    • while loop
    • for loop
    • for-each loop
  3. Jump statements
    • break statement
    • continue statement

Lets see the Decision making statements here.

if statements
Following are different types of if statement in Java.
Simple if statementif (condition) {    
    statement 1; //executes when condition is true
}    
if-else statementif (condition) {    
    statement 1; //executes when condition is true   
}  
else { 
    statement 2; //executes when condition is false   
}  
if-else-if ladderif (condition 1) {    
    statement 1; //executes when condition 1 is true   
}  
else if (condition 2) {  
    statement 2; //executes when condition 2 is true   
}  
else {  
    statement 2; //executes when all the conditions are false   
}  
Nested if-statementif(condition 1) {    
    statement 1; //executes when condition 1 is true   
    if(condition 2) {  
        statement 2; //executes when condition 2 is true   
    }  
    else {  
        statement 2; //executes when condition 2 is false   
    }  
}  
switch statement
    Switch statements are similar to if-else-if statements. The switch statement contains multiple blocks of code     called cases and a single case is executed based on the variable which is being switched. The switch     statement is easier to use instead of if-else-if statements. It also enhances the readability of the program.
    
    switch (expression) {  
        case value1:  
             statement1;  // Executed when expression== value1
             break;  
        case value2:  
             statement2;  // Executed when expression== value2
             break;  
        case value3:  
             statement3;  // Executed when expression== value3  
             break;  
        default:  
             default statement;    // Executed when expression is none of value1, value2 and value3
    }      
Iterative statements
Iterative statements are used to execute the block of code repeatedly while some condition evaluates to true. 

In Java, we have three types of loops that execute similarly. However, there are differences in their syntax and condition checking time.

  1. for loop
  2. for-each loop
  3. while loop
  4. do-while loop

Let's understand the loop statements one by one.

1. for loopThis loop is used when we exactly know the number of times, we want to execute the block of code. for(initialization; condition; increment/decrement) {    
    //block of statements
}    
2. for-each loopThis loop is used when we want to visit each element in a collection. There is no index in this loop. for(data_type var : array_name/collection_name) {
    //statements    
}
3. while loopThis loop is used if we don't know the number of iterations in advance. It is also known as the entry-controlled loop since the condition is checked at the start of the loop. If the condition is true, then the loop body will be executed; otherwise, the statements after the loop will be executed.
while(condition) {    
    //looping statements    
}
4. do-while loopThe do-while loop checks the condition at the end of the loop after executing the loop statements. When the number of iteration is not known and we have to execute the loop at least once, we can use do-while loop.
do {    
    //statements    
} while (condition);

Jump statements:
These statements are used to transfer the control of the program to the specific statements. In other words, jump statements transfer the execution control to the other part of the program. There are two types of jump statements in Java
  1. break
  2. break for labelled for loop
  3. continue
1. breakThis statement is used to break the current flow of the program and transfer the control to the next statement outside a loop or switch statement. However, it breaks only the inner loop in the case of the nested loop. It can only be written inside the loop or switch statement.
for(int j = 0; j<=15;j++) {    
    for (int k = 0; k<=20; k++) {    
        System.out.println(k);    
        if(k==5) {    
            break; /* This breaks the inner for loop and continues to next statement of outer for loop */
        }    
    }    
}    
2. break for a labelled loopIn Java, we can lable a loop. If the loops are labelled, break can be used to specifiy which loop should be broken.outer: for(int j = 0; j<=15;j++) {    
    inner: for (int k = 0; k<=20; k++) {    
        System.out.println(k);    
        if(k==5) {    
            break outer; /* This breaks the outer for loop and continues to next statement outside outer for loop */
        }    
    }    
} 
3. continueThe continue statement doesn't break the loop. It skips the specific part of the loop and jumps to the next iteration of the same loop immediately.
for(int i = 0; i<= 2; i++) {  
    for (int j = i; j<=5; j++) {  
        if(j == 3) {  
            continue; /* skips the inner for loop for j=3 only */  
        }  
        System.out.println(j);  
    }  
} 

Using Array
  • Arrays are used to store multiple values of same data type in a single variable, instead of declaring separate variables for each value. All elements in an array are stored in contiguous memory (consecutive memory locations). Individual elements can be accessed using the index of that element, starting from 0.
  • Arrays are objects in Java. They are always created on heap using new operator and referred by a reference which might be on stack. 

Single-Dimentional arrays
int [] arr;arr is a reference to an array which is not yet created.
arr= new int[3];    Here an array of 3 int is created on heap and it is now referred by arr (which is on stack)
arr[0]=10;
arr[1]=20;
arr[2]=30;​		Here value 10 is assigned to first member in the array, 20 is assigned to second member of array and value 30 is assigned to third member of the array. 
Note that, index of n'th member of an array is (n-1) because it starts from 0.
int size= arr.length;We can get size of an array using its property length.
int [] arr= {10,20,30};Here, the reference variable arr is created (on stack), array of 3 int is created (on heap) and the reference arr is set to refer the array. All done in single statement. 
int [] arr;
arr[0]=10;		This is wrong. arr, which is refernce of an array is created BUT the array is not yet created. Hence the reference is NOT initialized. Using such un-initialised reference gives null reference error. 
String[] arr= {"KNOW-IT", "C-DAC", "ACTS", "Pune"};arr is a reference to string array. An array of 4 string is created and arr is set to refer to that array. Using arr[0] gives "KNOW-IT", arr[1] gives "C-DAC", arr[2] gives "ACTS" and arr[3] gives "Pune".
arr.length is 4 in this case.
arr[arr.length-1]We can access last element in an array using arr[arr.length-1]
arr[arr.length]This will always give error, because there is no element present in an array whose index is equal to length. Note that index of first element is 0 and hence index of last element is lenght-1.

Multi-Dimentional arrays
We can create an array who every member itself is an array.  Such arrays are called as multi-dimentional arrays.
int arr[][]={{1,3,4},{3,4,5}}; Here arr is an array, whose first element itself is an array {1,3,4} and second member is also an array {3,4,5}

// Write 2 d arrays here
int arr[][]; // Line 1
arr= new int[3][3]; // Line 2
arr[0]= new int[3]; // Line 3
arr[1]= new int[3]; // Line 4
arr[0][0]=10; // Line 5
arr[1][0]=20; // Line 6		// Line 1: arr is a reference that can refer to a two-dimentional array of intergers. It is not yet initialized.
// Line 2: arr is not initiazed with an array of array of intergers. Note that every member of arr itself is a reference that can refer to another array.
// Line 3: arr[0] which is a reference to int array is now initialized with an an array of integers.
// Line 4: arr[1] which is a reference to int array is now initialized with an an array of integers.
// Line 5: First member of the first array is set to 10
// Line 6: First member of the second array is set to 20
arr.length // is 2
arr[0].length // is 3
arr[1].length // is 3
