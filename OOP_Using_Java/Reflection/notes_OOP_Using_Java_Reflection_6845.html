Reflection
Till now, we have seen how to write a class to model a real-life entity and then how to instantiate it in our program. Reflection is the process of knowing details about a class programatically.

Using reflection, we can programatically:
  1. find various data members, methods, constructor, super class of a class.
  2. Create instance of a class
  3. Invoke various methods in an instance. 

Reflection is an API that provide various classes and methods which can be used for performing the tasks mentioned above. These classes are present in packages java.lang and java.lang.reflect. The Reflection API is mainly used in:
  • IDE (Integrated Development Environment) e.g., Eclipse, MyEclipse, NetBeans etc.
  • Debugger
  • Test Tools etc.
class Class
"Class" is name of a class in reflection, that keeps all the information about an existing class, which can be retrieved at runtime programatically. This class is present in package java.lang.

In order to do reflection a Java class, we first need to create an object of Class. For example:
  • For doing reflection of a class Date in our program, we must first get the class object for Date class.
  • Similarly, for doing reflection of a class Person in our program, we must first get the class object for Person class. 

We can get class object for a class in following three ways:
  1. Using forName() method
  2. Using getClass() method
  3. Using .class extension

Lets see how this can be done.
1. Using forName() method
try  {
    Class clsPerson= Class.forName("ot.staff.Person");
} 
catch (ClassNotFoundException e) {
    System.out.println(e.getMessage());
}		Note that here clsPerson is class object for class ot.staff.Person.
Here, the forName() method takes the name of the class to be reflected as its argument. This class name should be fully qualified name of the class. This method throws checked exception as ClassNotFoundException
2. Using getClass() method on instance
try 
{
    Person p1= new Person("Sachin",15);
    Class clsPerson= p1.getClass();
} 
catch (Exception e) {
    System.out.println(e.getMessage());
}		Here, we are asking an existing instance of class Person to give it's Class object.
3. Using .class extension
try 
{
    Class clsPerson= Person.class;
} 
catch (Exception e) {
    System.out.println(e.getMessage());
}		We are getting Class object directly from class name using .class extension.
Once we get Class object (in of the three ways given above), it can be used further process of reflection. 
Finding class members
Following are various methods in class Class, which can be used to get complete information about a class.
Return typeMethod name   Description
ClassgetSuperclass()  

This method returns the Class which represents the superclass of the entity represented by this Class.
ObjectnewInstance()    This method creates a new instance of the class represented by this Class object.
PackagegetPackage()   It simply gets the package for this class.
intgetModifiers()   It returns the Java language modifiers for this class or interface, encoded in an integer.
Constructor[]getConstructors()Returns an array containing Constructor objects reflecting all the public constructors of the class represented by this Class object.
Method[]getMethods()  It returns an array containing Method objects reflecting all the public methods of the class or interface represented by this Class object. It also includes the declared methods of the class or interface and those inherited from super classes and super interfaces
Class[]getInterfaces()    It determines the interfaces implemented by the interface or class represented by this object.
Field[]getFields()    It returns an array containing Field objects reflecting all the accessible public fields of the interface or class represented by this Class object.

These methods from class Class can be used as follows to do reflection:
try 
{
    Class clsPerson= Person.class; // clsPerson is Class object for Person class

    Package p= clsPerson.getPackage(); // Find the package in which class Person is present
    String strClassName= clsPerson.getName(); // Get the fully qualified name of class Person
    Class clsSuperClass= clsPerson.getSuperclass(); // Get super class of class Person
    Constructor[] allConstructors= clsPerson.getConstructors(); // Get all constructors of class Person
    Method[] allMethods=  clsPerson.getMethods(); // Get all methods present in class Person
    Class[] allInterfaces= clsPerson.getInterfaces(); // Get all interfaces implemented by class Person
    Field[] allFields= clsPerson.getFields(); // Get all fields of class Person
    int iModifiers= clsPerson.getModifiers(); // Get all modifiers of class Person 
} 
catch (Exception e) {
    System.out.println(e.getMessage());
}
Creating instance dynamically
Using class Object, we can programatically create instance of a class as follows:
try 
{
    Class clsPerson= Person.class;
    Person p1 = (Person) clsPerson.newInstance();
    System.out.println(p1.getStrName());
} 
catch (Exception e) {
    System.out.println("Error"+e.getMessage());
}		Here, we call a method newInstance() on the class object to create instance of that class. This method return Object, so we must type-cast it appropriately.
Dynamic method invocation
Using class Object, we can programatically invoke a method as show below:
try 
{
    Class clsPerson= Person.class;
    Person p1 = (Person) clsPerson.newInstance();

     Class[] parameterType = new Class[] {String.class}; 
     Method m= clsPerson.getMethod("setName", parameterType); /* Here we have to pass name of method as first aragument and list of class objects of all parameters needed for this method */

     m.invoke(p1,"Virendra"); // We invoke method m on object p1 and pass "Virendra" as parameter

    // This is equivalent to p1.setName("Virendra");
} 
catch (Exception e) {
    System.out.println("Error"+e.getMessage());
}
