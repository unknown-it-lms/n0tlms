Command Line arguments in Linux

The detail of output we get from positional parameters are ::
Paramater       Description                                Value
     $0 name of script./script.sh
     $1first parameter passed.a
     $2second parameter passed.b
     $ {10}10 parameter passeduse bracket for double digit
if no bracket is used i.e. $10 then the output is a0
     $@lists all positional parameters, it sees parameters as different words.a b c d e f g h i j
     $*lists all parameter, it sees parameters as a single word if quoted ('$*'). a b c d e f g h i j
     $#number of parameter passed.10    $0 is the name of the shell.
     $$0Bashname of the shell
     $?Gives the exit status of most recently executed command.It gives 0 for successful, or an integer in the range 1 - 255 on error.
