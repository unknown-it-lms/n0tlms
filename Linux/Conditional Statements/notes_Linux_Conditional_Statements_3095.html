Notes: Linux- Conditional Statements

Conditional Statements
​​​​There are total 5 conditional statements which can be used in bash programming

if statement
if-else statement
if..elif..else..fi statement (Else If ladder)
if..then..else..if..then..fi..fi..(Nested if)
switch statement
if
if statement  
This block will process if specified condition is true.
Syntax 
if [ condition ]
then
   statement    
fi
​​​Implementing if statement  
​​​​​​​#Initializing two variables
a=10
b=20
if [ $a -eq $b ]                                 #Check whether they are equal  then
    echo "a is equal to b"
fi
if [ $a -ne $b ]                                 #Check whether they are not then equal echo "a is not equal to b"
fi
if...else
if-else statement  
If specified condition is not true in if part then else part will be execute.
​​​​​​​
Syntax 
if [ condition ]
then
   statement1
else
   statement2
fi
Implementing if statement 
#Initializing two variables
a=20
b=20
if [ $a -eq $b ]
then
    echo "a is equal to b"                      #If they are equal then print this
else
    echo "a is not equal to b"               #else print this
fi
if..elif..else..fi statement (Else If ladder)     
To use multiple conditions in one if-else block, then elif keyword is used in shell. If expression1 is true then it executes statement 1 and 2, and this process continues. If none of the condition is true then it processes else part.

Syntax 
if [ condition ]
then
   statement1
   statement2
   .
   .
elif [ expression2 ]
then
   statement3
   statement4
   .
   .
else
   statement5
fi      
nested if
if..then..else..if..then..fi..fi..(Nested if) 
Nested if-else block can be used when, one condition is satisfies then it again checks another condition. In the syntax, if expression1 is false then it processes else part, and again expression2 will be check.

Syntax 

​​​​​​​if [ condition ]
then
   statement1
   statement2
   .
else
   if [ condition ]
   then
      statement3
      .
   fi
fi
case
switch statement 
Case statement works as a switch statement if specified value match with the pattern then it will execute a block of that particular pattern. When a match is found all of the associated statements until the double semicolon (;;) is executed. A case will be terminated when the last command is executed. If there is no match, the exit status of the case is zero.
Syntax 
case  $var  in
   Pattern 1) Statement 1;;
    :
    :
   Pattern n) Statement n;;
esac
   
Implementing switch statement 
CARS="bmw"
case   "$CARS"   in                                                                                           #Pass the variable in string
    "mercedes") echo "Headquarters - Affalterbach, Germany" ;;           #case 1
    "audi") echo "Headquarters - Ingolstadt, Germany" ;;                         #case 2  
    "bmw") echo "Headquarters - Chennai, Tamil Nadu, India" ;;            #case 3
esac
Output
Headquarters - Chennai, Tamil Nadu, India.

Note: Shell scripting is a case-sensitive language, which means proper syntax has to be followed while writing the scripts.
