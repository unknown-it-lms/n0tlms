Factor Analysis
​​​​​​​Factor analysis is a statistical technique used to uncover underlying patterns (factors) in a set of observed variables. It helps reduce data complexity, discover latent variables, and aid in data interpretation. The process involves extracting factors, rotating them for clarity, and interpreting the results based on factor loadings. Factor analysis is valuable for dimensionality reduction and understanding relationships between variables.
Overview of Factor Analysis

Factor analysis is a statistical technique used to uncover underlying patterns (factors) in a set of observed variables. It aims to identify the latent (hidden) variables that contribute to the observed correlations or covariances among the variables. This technique is widely used in fields such as psychology, sociology, marketing, and finance to reduce data complexity, discover underlying structures, and aid in data interpretation. An in-depth explanation of factor analysis:

1. Assumptions of Factor Analysis:
  • The observed variables are assumed to be influenced by a smaller number of underlying factors.
  • The factors are latent and cannot be directly measured but can be inferred from the observed variables.
  • The observed variables are linearly related to the underlying factors.
  • The errors or unique variances of the observed variables are uncorrelated with each other.
2. Steps in Factor Analysis:

  • Data Collection: Gather data on a set of variables from a sample of subjects or observations.

  • Correlation Matrix or Covariance Matrix: Calculate the correlation matrix if the variables are on the same scale, or the covariance matrix if the variables are on different scales. The matrix represents the pairwise correlations or covariances between all observed variables.

  • Factor Extraction: Determine the number of factors to extract. Common methods include Kaiser's criterion (eigenvalue greater than 1), scree plot examination, and parallel analysis. Factor extraction transforms the original variables into a set of uncorrelated linear combinations called "factor scores."

  • Factor Rotation: After extraction, the factors are often rotated to simplify interpretation and enhance clarity. Common rotation methods include orthogonal rotation (e.g., Varimax) and oblique rotation (e.g., Promax). Orthogonal rotation assumes that the factors are uncorrelated, while oblique rotation allows for correlations between factors.

  • Factor Loading Estimation: Estimate the factor loadings, which represent the strength and direction of the relationship between each variable and the underlying factors. High factor loadings indicate that the variable is strongly associated with a particular factor.

  • Factor Interpretation: Interpret the extracted and rotated factors based on the patterns of factor loadings. Assign meaningful labels or names to each factor that captures its underlying meaning.

  • Factor Scores Calculation: Calculate factor scores for each subject or observation based on their responses to the observed variables. Factor scores represent the degree to which each individual is associated with each factor.

3. Types of Factor Analysis:

  • Exploratory Factor Analysis (EFA): In EFA, the number of factors to extract is not predetermined, and the analysis aims to explore the underlying factor structure of the data.

  • Confirmatory Factor Analysis (CFA): In CFA, the number of factors and their relationships are pre-specified based on theoretical or empirical grounds. CFA is used to test specific hypotheses about the factor structure.

4. Interpretation of Factor Analysis Results:
  • Factor Loadings: High loadings indicate a strong association between a variable and a factor.
  • Communalities: The proportion of variance in each variable explained by the extracted factors.
  • Eigenvalues: Eigenvalues represent the variance explained by each factor.
  • Factor Matrix: A matrix displaying the factor loadings after rotation.

Factor analysis is a valuable tool for dimensionality reduction, identifying underlying structures in data, and generating new variables (factor scores) that can be used in subsequent analyses. However, it requires careful consideration of data suitability, the number of factors to extract, and the appropriate rotation method to ensure meaningful and valid results.
