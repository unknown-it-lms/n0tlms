Notes: Cassandra- Alter table

Alter table
​​​​​​​Alter in Cassandra is used same as in RDBMS to change the table sturcture.
Adding new column 
Syntax
ALTER TABLE table_name
ADD new_column datatype;
eg
alter table emp
add email string;
Deleting the column
Syntax
ALTER TABLE table_name
DROP COLUMN column_name;
eg
ALTER TABLE emp
DROP COLUMN mgr;
Renaming Column
Can rename only PK.
Syntax
ALTER TABLE table_name
RENAME column_name to new_column_name;
eg
alter table mytab
rename num to find;
Delete Table
Truncate will delete the table as well as the structure.
Syntax
TRUNCATE  table_name;
eg
TRUNCATE emp;
Createing Index
Syntax
CREATE INDEX idx_name
ON TABLE(column_name);
eg
create index idx_nm
on student(name);
Deleting Index
Syntax
DROP INDEX index_name;
eg
DROP INDEX idx_nm;
