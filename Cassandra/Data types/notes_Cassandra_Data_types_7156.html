Data types

Collections
Cassandra Query Language also provides a collection data types. The following table provides a list of Collections available in CQL.   
CollectionDescription
list 
 A list is a collection of group of  elements, ordered , of same data type and duplicates are allowed.
map
 A map is a collection elements with data stored as a  key-value pairs. Here key is unique.
set 
A set is a collection of  group of  elements, unique, unordered and of same data type.
tupleA set of fixed length values of different data types and duplicates are allowed.
