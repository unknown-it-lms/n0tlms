Notes: Cassandra- Cassandra shell commands

Cassandra shell commands
Cqlsh 
Cassandra provides a prompt Cassandra query language shell (cqlsh) that allows users to communicate with it. Using this shell, a user can execute Cassandra Query Language (CQL).

Using cqlsh  user can
>define a schema
>insert data
>execute a query
 Shell Commands  
CommandDiscription
helpCassandra Help command is used to show the complete commands details of Cassandra.
clearCassandra Clear command will clear all previous output from the screen as mentioned in the below example.
captureCassandra Capture command will capture the output of the command and put it in a file.
consistencyCassandra Consistency command is used to check how many nodes in replica will respond to the coordinator node for a task. The consistency level can be changed using this command.
copy to Cassandra Copy To Command copies data from the Cassandra table to a file.
copy fromCopy From Command copies data from the files to the Cassandra table.
describe table /tablesCassandra Describe Tables or Table commands are used to show the detail about the tables.
describe keyspacesCassandra Keyspaces command shows the detail about all the Keyspaces present in the Cassandra Cluster.
expandThe Cassandra Expand command expands the output of a query in a more readable form.
showCassandra Show command displays the information about the current session, host detail, Version information, and so on.
sourceWe can use the Cassandra Source command to execute multiple commands from a file
