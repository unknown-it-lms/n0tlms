Tuple Concept and Basic Operation
A tuple is a collection which is ordered and unchangeable. In Python tuples are written with round brackets.
Tuples are sequences, just like lists. The differences between tuples and lists are, the tuples cannot be changed unlike lists and tuples use parentheses, whereas lists use square brackets. 

Creating tuple:
Code:Output:
tup1 = ('physics', 'chemistry', 1997, 2000)
tup2 = (1, 2, 3, 4, 5 )
tup3 = "a", "b", "c", "d"

note: you can define tuple without using () also.

The empty tuple is written as two parentheses containing nothing -
Code:Output:
tup1 = ()

To write a tuple containing a single value you have to include a comma, even though there is only one value -
Code:Output:
tup1 = (50,)

Like string indices, tuple indices start at 0, and they can be sliced, concatenated, and so on.

Updating Tuples
Tuples are immutable which means you cannot update or change the values of tuple elements.

Deleting Tuple
Unlike lists, the tuple items cannot be deleted by using the del keyword as tuples are immutable. To delete an entire tuple, we can use the del keyword with the tuple name.

Built-in Tuple Functions
  1.  len(tuple)    It calculates the length of the tuple.
  2.  max(tuple)    It returns the maximum element of the tuple
  3.  min(tuple)    It returns the minimum element of the tuple.
  4. tuple(seq)    It converts the specified sequence to the tuple.
  5. sorted(tuple): Returns a sorted version of a tuple, in ascending order by default.
