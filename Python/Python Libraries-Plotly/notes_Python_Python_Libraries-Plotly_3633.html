Python Libraries-Plotly
​​​​​​​Plotly is a Python graphing library that makes interactive, publication-quality graphs online. It provides a wide range of chart types and customization options, and it supports interactive features such as zooming, panning, and hovering. Here's an overview of some key features and common tasks you can perform with Plotly:
Introduction
  1. The Plotly Python library is an interactive open-source library.
  2. This can be a very helpful tool for data visualization and understanding the data simply and easily.
  3. plotly graph objects are a high-level interface to plotly which are easy to use.
  4. It can plot various types of graphs and charts like scatter plots, line charts, bar charts, box plots, histograms, pie charts, etc.

Package Structure of Plotly
​​​​​​​There are three main modules in Plotly. They are:
  1. plotly.plotly
  2. plotly.graph.objects
  3. plotly.tools

  1. plotly.plotly acts as the interface between the local machine and Plotly. It contains functions that require a response from Plotly's server.
  2.  plotly.graph_objects module contains the objects (Figure, layout, data, and the definition of the plots like scatter plot, line chart) that are responsible for creating the plots.
  3.  The Figure can be represented either as dict or instances of plotly.graph_objects.
  4. The figures created, manipulated and rendered by the plotly Python library are represented by tree-like data structures which are automatically serialized to JSON for rendering by the Plotly.js JavaScript library.
Different Types of Charts
​​​​​​​1. Scatter Chart
Code:Output:
import plotly
import numpy as np
import plotly.graph_objects as go

xdata= np.random.randint(1,30,100)
ydata= np.random.randint(1,30,100)
fig=go.Figure(data=go.Scatter(x=xdata,y=ydata,mode="markers"))
fig.show()
​

2. Bar Chart
Code:Output:
import plotly
import plotly.graph_objects as go

cnt= ["india","Pak","nepal","Bhutan","Mexico","Texas"]
ydata= [80,23,45,67,89,48]
fig=go.Figure([go.Bar(x=cnt,y=ydata)])
fig.show()
