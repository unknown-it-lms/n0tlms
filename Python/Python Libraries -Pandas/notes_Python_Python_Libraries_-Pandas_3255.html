Python Libraries -Pandas
​​​​​​​Pandas is a popular Python library for data manipulation and analysis. It provides powerful data structures and functions to work with structured data, making it an essential tool for data scientists, analysts, and engineers. Here's an overview of some key features of Pandas:
Basics of Pandas
  1. Pandas is defined as an open-source library that provides high-performance data manipulation in Python.
  2. The name of Pandas is derived from the word Panel Data, which means an Econometrics from Multidimensional data.
  3. It is used for data analysis in Python and developed by Wes McKinney in 2008.
  4. Pandas is built on top of the Numpy package, means Numpy is required for operating the Pandas.
data analysis requires lots of processing, such as restructuring, cleaning or merging, etc. There are different tools are available for fast data processing, such as Numpy, Scipy, Cython, and Panda. But we prefer Pandas because working with Pandas is fast, simple and more expressive than other tools.

Benefits of Pandas
  1. Data Representation: It represents the data in a form that is suited for data analysis through its DataFrame and Series.
  2. Clear code: The clear API of the Pandas allows you to focus on the core part of the code. So, it provides clear and concise code for the user.
Python Pandas Data Structure
The Pandas provides two data structures for processing the data, i.e., Series and DataFrame, which are discussed below:

1) Series
  • It is defined as a one-dimensional array that is capable of storing various data types.
  • The row labels of series are called the index.
  • We can easily convert the list, tuple, and dictionary into series using "series' method.
  • A Series cannot contain multiple columns. It has one parameter.
Creating Series from Array:
Before creating a Series, Firstly, we have to import the numpy module and then use array() function in the program.

Python Pandas DataFrame
It is a widely used data structure of pandas and works with a two-dimensional array with labeled axes (rows and columns). DataFrame is defined as a standard way to store data and has two different indexes, i.e., row index and column index.
It consists of the following properties:
  • The columns can be heterogeneous types like int, bool, and so on.
  • It can be seen as a dictionary of Series structure where both the rows and columns are indexed. It is denoted as "columns" in case of columns and "index" in case of rows.

Pandas Operation on Series
  • data: It can be any list, dictionary, or scalar value.
  • index: The value of the index should be unique and hashable. It must be of the same length as data. If we do not pass any index, default np.arrange(n) will be used.
  • dtype: It refers to the data type of series.
  • copy: It is used for copying the data.

Importing Data
  1. pd.read_csv(filename) : It read the data from CSV file.
  2. pd.read_table(filename) : It is used to read the data from delimited text file.
  3. pd.read_excel(filename) : It read the data from an Excel file.
  4. pd.read_sql(query,connection _object) : It read the data from a SQL table/database.
  5. pd.read_json(json _string) : It read the data from a JSON formatted string, URL or file.
  6. pd.read_html(url) : It parses an html URL, string or the file and extract the tables to a list of dataframes.
  7. pd.read_clipboard() : It takes the contents of clipboard and passes it to the read_table() function.
  8. pd.DataFrame(dict) : From the dict, keys for the columns names, values for the data as lists.

Filter, Sort, and Groupby
  1. df[df[col] > 0.5]: Returns the rows where column col is greater than 0.5
  2. df[(df[col] > 0.5) & (df[col] < 0.7)] : Returns the rows where 0.7 > col > 0.5
  3. df.sort_values(col1) :It sorts the values by col1 in ascending order.
  4. df.sort_values(col2,ascending=False) :It sorts the values by col2 in descending order.
  5. df.sort_values([col1,col2],ascending=[True,False]) :It sort the values by col1 in ascending order and col2 in descending order.
  6. df.groupby(col1): Returns a groupby object for the values from one column.
  7. df.groupby([col1,col2]) :Returns a groupby object for values from multiple columns.
  8. df.groupby(col1)[col2]) :Returns mean of the values in col2, grouped by the values in col1.
  9. df.apply(np.mean) :Its task is to apply the function np.mean() across each column.
  10. nf.apply(np.max,axis=1) :Its task is to apply the function np.max() across each row.

Data Cleaning
  1. df.columns = ['a','b','c']: It rename the columns.
  2. pd.isnull(): It checks for the null values and returns the Boolean array.
  3. pd.notnull(): It is opposite of pd.isnull().
  4. df.dropna(): It drops all the rows that contain the null values.
  5. df.dropna(axis= 1): It drops all the columns that contain null values.
  6. df.dropna(axis=1,thresh=n): It drops all the rows that have less than n non null values.
  7. df.fillna(x): It replaces all null values with x.
  8. s.fillna(s.mean()): It replaces all the null values with the mean(the mean can be replaced with almost any function from the statistics module).
  9. s.astype(float): It converts the datatype of series to float.
  10. s.replace(1, 'one'): It replaces all the values equal to 1 with 'one'.
  11. s.replace([1,3],[ 'one', 'three']):It replaces all 1 with 'one' and 3 with 'three'.
  12. df.rename(columns=lambda x: x+1):It rename mass of the columns.
  13. df.rename(columns={'old_name': 'new_ name'}): It consist selective renaming.
  14. df.set_index('column_one'): Used for changing the index.
  15. df.rename(index=lambda x: x+1): It rename mass of the index.

It also ignores non-numeric data type columns from the DataFrame.
