Notes: Python- Python Modules

Python Modules
  • ​​​​​​​Python modules are files containing Python code that can be imported and used in other Python programs.
  • They allow you to organize code into reusable units, making your programs more modular and easier to manage.
  • Modules can include functions, classes, and variables.
Module
​​​​​​​A Python module is a file containing Python definitions and statements. A module can define functions, classes, and variables. A module can also include runnable code. Grouping related code into a module makes the code easier to understand and use. It also makes the code logically organized.
  1. Modules refer to a file containing Python statements and definitions.
  2. A file containing Python code, for example: example.py, is called a module, and its module name would be example.
  3. We use modules to break down large programs into small manageable and organized files. Furthermore, modules provide reusability of code.
  4. We can define our most used functions in a module and import it, instead of copying their definitions into different programs.
Create a Module
To create a module, you simply create a Python file with a .py extension and write your code inside it.
​​​​​​​This code can contain function and class definitions, as well as variable assignments.
Code:Output:
# Python Module example
def add(a, b):
   """This program adds two numbers and return the result"""
   result = a + b
   return result

Here, we have defined a function add() inside a module named example (save file as example.py). The function takes in two numbers and returns their sum.
Use a Module
Once you've created a module, you can import it into other Python scripts using the import statement. For example, if you have a module called example.py, you can import it like this:
Syntax:
Code:Output:
import module_name

Example:
Code:Output:
import example

Note: This does not import the functions or classes directly instead imports the module only. To access the functions inside the module the dot(.) operator is used.

Using the module name we can access the function using the dot . operator. For example:
Code:Output:
example.add(4,5.5)
9.5

Importing file from another folder:
There are a few ways to import a Python file from another folder. 
One way is to use the sys.path.append() function.
 This function adds the path to the folder containing the file to the Python path, which is a list of directories that Python searches for modules and packages. 
Once you have added the path to the Python path, you can then use the import statement to import the file as if it were in the current directory.
For example
if you have a file called myfile.py in a folder called my_folder, you could import it as follows:
​​​​​​​
Code:Output:
import sys
sys.path.append('my_folder')


import myfile

Naming a Module
  1. ​​Module Aliases:
    You can also alias a module when importing it, which can be useful for modules with long names or to avoid naming conflicts.
    For example:
    Code:Output:
    import my_module as mm

    Then you can use mm instead of my_module in your code.
    ​​​​​​​
  2. Importing Specific Items:
    If you only need specific functions or classes from a module, you can import them directly instead of importing the entire module.
    syntax:
    Code:Output:
    from my_module import my_function, MyClass


    For example:​​​​​​
    Code:Output:
    # importing sqrt() and factorial from the module math
    from math import sqrt, factorial

    # if we simply do "import math", then math.sqrt(16) and math.factorial() are required.

Import From Module
Python’s from statement lets you import specific attributes from a module without importing the module as a whole.
You can choose to import only parts from a module, by using the from keyword.
Code:Output:
# importing sqrt() and factorial from the module math
from math import sqrt, factorial

# if we simply do "import math", then math.sqrt(16) and math.factorial() are required.
print(sqrt(16))
print(factorial(6))
Note: When importing using the from keyword, do not use the module name when referring to elements in the module. Example: use sqrt(16)  and not math.sqrt(16) .

Date Module
Importing the datetime Module:
import datetime
Core Classes:
  • date: Represents a specific date (year, month, day) without time information.
  • time: Represents a specific time (hour, minute, second, microsecond) without date information.
  • datetime: Combines both date and time information.
  • timedelta: Represents a duration (difference) between two dates or times.

Creating Date and Time Objects:

date:

Math Module

The math module in Python provides various mathematical functions for performing mathematical operations. It includes functions for basic arithmetic operations, trigonometric functions, logarithmic functions, and more. Here's an overview of some of the commonly used functions and constants in the math module:

  1. Constants:

    • math.pi: Represents the mathematical constant π (pi).
    • math.e: Represents the mathematical constant e, the base of natural logarithms.

  2. Basic arithmetic functions:

    • math.sqrt(x): Returns the square root of x.
    • math.pow(x, y): Returns x raised to the power y.
    • math.exp(x): Returns e raised to the power x.
    • math.log(x[, base]): Returns the natural logarithm of x (base e), or the logarithm of x to the given base if specified.

  3. Trigonometric functions:

    • math.sin(x), math.cos(x), math.tan(x): Returns the sine, cosine, and tangent of x (in radians).
    • math.asin(x), math.acos(x), math.atan(x): Returns the inverse sine, cosine, and tangent of x, respectively.

  4. Hyperbolic functions:

    • math.sinh(x), math.cosh(x), math.tanh(x): Returns the hyperbolic sine, cosine, and tangent of x.

  5. Angular conversion functions:

    • math.degrees(x): Converts angle x from radians to degrees.
    • math.radians(x): Converts angle x from degrees to radians.

  6. Other functions:

    • math.ceil(x): Returns the smallest integer greater than or equal to x.
    • math.floor(x): Returns the largest integer less than or equal to x.
    • math.factorial(x): Returns the factorial of x.
    • math.gcd(a, b): Returns the greatest common divisor of integers a and b.

Code:Output:
import math

# Constants
print("Value of pi:", math.pi)

# Basic arithmetic functions
print("Square root of 25:", math.sqrt(25))
print("2 raised to the power 3:", math.pow(2, 3))
print("Natural logarithm of 10:", math.log(10))

# Trigonometric functions
print("Sine of 0:", math.sin(0))
