Lists
HTML Lists are used to specify lists of information. All lists may contain one or more list elements. There are three different types of HTML lists
  1. Unordered List or Bulleted List (ul)
  2. Ordered List or Numbered List (ol)
  3. Description List or Definition List (dl)
Unordered list
All the list items are marked with bullets. It is also known as bulleted list also. The Unordered list starts with <ul> tag and list items start with the <li> tag.
To change the type of bullet, attribute ‘type’ can be used which can take the values as ‘disc’, ‘circle’ or ‘square’.
HTMLOuput
<ul>
    <li> Beetroot</li>
    <li> Ginger </li>
    <li> Potato</li>
    <li> Radish </li>
  </ul>		​
<ul type="square">
    <li> Beetroot</li>
    <li> Ginger </li>
    <li> Potato</li>
    <li> Radish </li>
  </ul>		​
Ordered lists

  1. type : To change the type of numbering, attribute ‘type’ can be used which can take the values as 1, i, I, a, A. 
  2. start : To change numbering of first list element means list will not begin with number 1.

The HTML definition list contains following three tags: <dl> tag defines the start of the list. <dt> tag defines a term. <dd> tag defines the term definition (description).

Creating tables
In HTML, tables can be created in the webpage using the <table> tag in support with the <tr>, <td> and <th> tags. The HTML tables allow displaying the data (e.g. image, text, link) in columns and rows of cells.
A simple table can be created using <table> </table>.
After creating an HTML table, you should add a border to it, as borders are not added by default.
<table border="1"> </table>
By default separate border gets added to cells and entire table. If 2 borders should be merged together to get a single line border :
<table border="1" style="border-collapse:collapse">
Important table attributes 
  1. border : The border attribute in HTML is used to specify the width of the border around a table and its cells. It's a simple way to add borders to tables without using CSS. Border attribute is deprecated in HTML5 and and table borders should be managed using CSS.
  2. cellpadding  : This attribute defines the space between the content of a cell and its border
  3. cellspacing  : This attribute defines the size of the space between two cells in a percentage value or pixels
  4. rowspan : It is possible to extend rows and columns of a table across many other rows and columns. By default cell will span across inly one row and one column. This attribute is used to make a cell span more than one rows
  5. colspan : This attribute is used to make a cell span more than one columns.
  6. width : This attribute is used to modify the width of entire column.
  7. height : This attribute is used to modify the heightof entire row.

HTML Frames
HTML frames are used to divide the browser window into multiple independent sections, each displaying a separate HTML document. They are commonly used earlier to create layouts with multiple panes, such as a navigation frame and a content frame.
Frames are created with <frameset> and <frame> elements. 
  1. frameset : This element is used to decide the layout of a page whether frames to be arranged vertically or horizotally. It has attributes like rows or cols which takes the size of the frame as it's value. If rows is used frames are arranged in horizontal manner and cols is used, frames are arranged in vertical manner.
  2. frame : This element defines each individual frame. This element has 2 important attributes maninly name and src.
    1. name : It is used to give a name to the frame, which can be used as a target for link.
    2. src : It is used to specify the URL of the web page which is to be displayed in the document. 
HTMLOutput
<html>
<head>
  <title>Web Page with Frames </title> 
</head>
<frameset cols="30%,*">
    <frame src="header.html" name="frame1" />
    <frame src="body.html" name="frame2" />
</frameset>
</html>

Nested frames 
As the name indicate nested frame is frame with another frame. This can be nested in any level.
HTMLOutput
<html>
<head>
  <title>Web Page with Frames </title>
</head>
<frameset rows="50%,50%">
    <frame src="header.html" name="frame1" />
    <frameset cols="40%,60%">
        <frame src="frameA.html" name="frame2" />
        <frame src="frameB.html" name="frame3" />
    </frameset>
</frameset>
</html>		​
Displaying content in adjacent frame 
The frame can be used as a target of hyperlink displayed in that page. This is possible by defining target attribute for anchor element. This target attribute can have the value specified as the name given to a frame using name attribute. After clicking the hyperlink the destination page will be opened in the frame which has the name mentioned in the target attribute.
‘target’ attribute can take following values.
ValueDescription
_blankOpens the linked document in a new window or tab
_selfOpens the linked document in the same frame as it was clicked (this is default)
_parentOpens the linked document in the parent frame
_topOpens the linked document in the full body of the window
framenameOpens the linked document in a named frame


Need of iframe
There are some disadvantages of using frames.
  • Frameset can not have their own content as body tag is removed from frameset.
  • Bookmarks only bookmark the top level pages.
  • Search engines can not use all the pages in web page with frames for searching the content.
  • All the modern browsers may not support frames.
Because of these disadvantages, frames are removed from HTML 5 and have been replaced by iframes. Iframes means inlined frames which can be shown in the existing content of web page. Frames always need a parent as frameset but iframe does not need any specific parent element.
  1. You can define an inline frame with HTML tag <iframe>. The <iframe> tag can appear anywhere in your document.
  2. The <iframe> tag defines a rectangular region within the document in which the browser can display a separate document, including scrollbars and borders.
  3. This inline frame is used to embed another document within the current HTML document.
  4. The src attribute is used to specify the URL of the document that occupies the inline frame.
HTMLOutput
<body>
      <p>Document content goes here...</p>      
      <iframe src="mypage.html" width="200" height="200">
           Sorry your browser does not support inline frames.
      </iframe>      
      <p>Document content also go here...</p>
 </body>   
​
Name attribute
The name attribute in an <iframe> element specifies a name for the inline frame (iframe). This name can be used as a target for hyperlinks.
