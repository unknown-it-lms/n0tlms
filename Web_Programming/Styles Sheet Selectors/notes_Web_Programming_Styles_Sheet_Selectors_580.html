Styles Sheet Selectors
In this chapter we wil dive deep into CSS and learn more about different types of selectors that are available to define style rules. We will learn about CSS box model and frequently required designs for web pages in the form of columnar output etc.
Different CSS Selectors
There are different types of selectors that are available which are based on name of tag, value of class attribute, value of id attribute, parent-child relationship, pseudo selectors etc.
Type of selectorUsageDescription
Element Type Selectorsp { color: green; }
/* All paragraphs will get a text color as green. */		It is the most basic way of selecting the HTML elements by using tag names.
Universal Selector* { border: 1px solid blue;}
/* Every element will get a solid 1px wide border*/		Symbol used is * and it maches every element on the web page.
Class Selectors.highlight { background-color: gray; }
/*  All the elements having class attribute specified as "highlight", will get displayed with gray background color. */		Symbol used is .(dot) followed by class attribute value and it matches all the elements that has the specified class.
p.highlight { background-color: gray; }
/* All the paragraph elements having class attribute specified as "highlight", will get displayed with gray background color. */
Id Selectors#content { margin: 5px }
/* Element having id attribute specified as "content", will get 5px margin from all sides. */		Symbol used is #, followed by id attribute value and it matches all the elements that has the specified id.
Group selectorh1, h2, h3, h4, h5 { text-align: center }
/* all headings from h1 to h5 will get center text-alignment */		Symbol used is ,(comma) for grouping multiple elements together
Selection based on attributes[target='_blank'] { text-decoration: none}
/* All elements where target attribute is specified with the value as '_blank' will get text-decoration as "none" */		Symbol used is [](square brackets) with attribute-value pair specified
Selecting children elementsdiv p { color: blue}
/* selects all P elements which are any child(descendant) of div element */		two separate selectors, separated by a space.
Selecting direct child elementdiv > p {color: blue}
/* selects all P elements which are immediate child of div element */		two separate selectors, separated by a '>' (greater than).
Selecting next siblingdiv + p {color: blue}
/* selects all P elements which are next sibling  of div element */		two separate selectors, separated by a '+' (plus).
Pseudo selectorsbutton:hover {
  background-color: blue;
}
/*When mouse pointer is over the button element, changes it's background color to blue*/		keywords used to select elements based on their state or position in the web page
input:focus {
  border-color: red;
}
/* When input element is focused, it will get red border color*/
input[type="checkbox"]:checked {
  background-color: green;
}
/* When checkbox is checked, it will get green background-color */
p:first-child {
  font-weight: bold;
}
/* Paragraph element which is first child of its parent will be displayed with bold effect  */
a:visited    {
  color: blue
}      
 /* visited links will get blue color       */

Page or element backgrounds images
Managing backgrounds using CSS involves styling background properties of HTML elements to display the web page in a better fashion. Following CSS properties are used for managing image background :
PropertyUsageDescription
background-image.header {
    background-image: url('background.jpg');
}		Sets an image as the background of an element.
background-repeat.content {
    background-image: url('background.png');
    background-repeat: repeat;
}		Specifies how a background image should repeat.
background-position.hero {
    background-image: url('background.jpg');
    background-position: center;
}		Sets the starting position of a background image.
background-attachment.parallax {
    background-image: url('background.jpg');

}		A shorthand property for setting all background properties in one declaration
Managing spaces around elements
Each element on a web page can be considered as a box. Every box is composed of four parts (or areas), defined by their respective edges: the content edge, padding edge, border edge, and margin edge.​​​​​​​ The actual content of the box can be text or images. It's the innermost part of the box and is surrounded by padding, border, and margin. Padding is the space between the content and the border. The border surrounds the padding. Margin is the space between the border and neighboring elements. Padding, border and margin can be individually set for 4 sides or can be set for all sides in short notation.
​
HTMLDescription
 <a href="details.html" style="padding: 20px;border: 2px solid black">
        More details
 </a>
/* 20px space is left inside the border from all sides from the content */     		​
<a href="details.html" style="padding-left: 20px; padding-right: 30px;border: 2px solid black">
        More details
 </a>
/* 20px space is left inside the border from left side and 30 px space is left on the right side of the content */ 		​
<a href="details.html" style="margin: 20px; border: 2px solid black">
        More details
    </a>
/* 20px space is left outside the border from all sides from the content */   		​
<a href="details.html" style="margin-top: 20px; margin-bottom: 30px; border: 2px solid black">
        More details
    </a>
/* 20px space is left outside the border from top side and 30 px space is left on the bottom side of the content */ 		​
Columnar output
Suppose there is a need of a layout where you want to float two elements side by side, and then add a third element below them, float and clear CSS properties will help for this kind of layout. Using the float and clear properties in CSS is a common technique for building such layouts, especially for displaying content in multiple columns. The float CSS property places an element on the left or right side of its container. Elements next to a floating element will flow around it
HTML
<head>
    <style>
        .column{
            float: left;
            width: 30%; /* Each column takes up 30% of the container width */
            background-color: lightblue;
            padding: 10px;
            border: 1px solid black;  
        }
        .container{
            max-width: 1300px;
        }
        .clearfix {
            clear: left
        }
    </style>
</head>
<body>
    <div class="container">
        <div class="column">Column 1</div>
        <div class="column">Column 2</div>
        <div class="column">Column 3</div>
        <div class="clearfix">Column below</div>
    </div>
</body>

Layering elements on one another
Using positioning and the z-index property in CSS allows to display overlapping content on the web page. The position property in CSS is used to specify the positioning method of an element relative to its containing element. Allowed values are static(default), relative, fixed, absolute or sticky. The z-index property specifies which element to be placed in front or behind other elements when they are overlapped on one another. It can have any positive or negative value. Elements with a higher z-index value appear above elements with a lower z-index value.
HTMLOutput
<head>
    <style>
        .box {
            position: absolute;
            width: 100px;
            height: 100px;
            padding: 10px;
        }
        .red {
            background-color: red;
            top: 20px;
            left: 20px;
            z-index: 3;
        }
        .green {
            background-color: green;
            top: 40px;
            left: 40px;
            z-index: 2;
        }
        .blue {
            background-color: blue;
            top: 60px;
            left: 60px;
            z-index: 1;
        }
    </style>
</head>
<body>
    <div class="container">
        <div class="box red">Red Box</div>
        <div class="box green">Green Box</div>
        <div class="box blue">Blue Box</div>
    </div>
</body>		​
