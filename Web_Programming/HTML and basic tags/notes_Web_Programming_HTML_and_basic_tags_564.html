Introduction and Structure of HTML
Introduction to HTML
HTML (HyperText Markup Language) is the standard markup language for creating web pages. It's used to structure content and specify its presentation on a web page with the help of different standard tags available in HTML.
Structure of HTML
​​​​​​​The basic structure of an HTML document consists of 5 elements:
  1. <!DOCTYPE>
  2. <html>
  3. <head>
  4. <body>
  5. <title>
  6. <meta>

The DOCTYPE
A DOCTYPE declaration must be specified on the first line of each web document. The DOCTYPE tells the web browser which version of HTML the page is written in.  <!DOCTYPE html> declaration indicates that we are following HTML 5 version. It is not compulsory to write this declaration

The <html> Element
Immediately following the DOCTYPE declaration is the <html> element:
The <html> element tells the browser that the page will be formatted in HTML and, optionally, which world language the page content is in.

The <head> Element
The <head> element contain most of the elements which do not get displayed directly on the web page. But elements in the head section help the browser in understanding the requirements of the current web page. Head element may consist of following elements:
The <title> element
 The <title> element defines the text to show in the web browser's title bar.
The <meta> Element
<meta> tag is used to provide metadata about the HTML document. Metadata includes information such as character set, author, description, keywords, and more.
There can be multiple <meta> tags in the same web page.  We will use <meta> element when we use bootstrap for the web pages.

The <body> Element
The <body> element contains elements which are used to display actual content (text, images, videos, links, etc.) to the end user. Elements written in body section (means body element) are directly rendered in web browser.
Example :
<!DOCTYPE html>
<html>
<head>
     <title>My First Web Page</title>
     <meta name="description" content="Description of the page">
     <meta name="author" content="Author's Name"> 
</head>
<body>
     Web Page content goes here.
</body>
</html>
Note : Every web document must include one and only one instance of DOCTYPE, <html>, <head>, <body>, and <title>.
HTML Basic Tags
There are few elements that every web page needs to make use of. We will see details of such basic elements
Heading tags :
A HTML heading or HTML h tag can be defined as a title or a subtitle which you want to display on the web page. When you place the text within the heading tags <h1>.........</h1>, it is displayed on the browser in the bold format and size of the text depends on the level of heading.
There are six different HTML headings which are defined with the <h1> to <h6> tags. Heading elements (h1....h6) should be used for headings only. They should not be used just to make text bold or big.
Heading tags are very important tags that are searched by search engines.

Paragraph tag
The <p> tag defines a paragraph of text. It is a block-level element and always starts on a new line. Before and after each paragraph, browsers add margin automatically.

<body>

    <p> This is second paragraph </p>
    <p> This is third paragraph </p>
</body>		​
Line breaking tag
Browser by default trims all the spaces and converts it to single space. Whenever you use the <br /> element, anything written after that, starts from the next line. This tag is an example of an empty element, where you do not need opening and closing tags, as there is nothing to go in between them.

Anchor tag
The HTML anchor tag defines a hyperlink that links one page to another page.  The "href" attribute is the most important attribute of the HTML anchor tag. The href attribute is used to define the address of the file to be linked. In other words, it points out the destination page. The syntax of HTML anchor tag is given below.   
 <a href = "..........."> Link Text </a>
Appearance of HTML anchor tag : 
  • A hyperlink is displayed underlined and blue.     
  • When mouse pointer is on the hyperlink text, it changes to hand.   
HTMLOutput
<body>
    <a href="second.html"> Click for Second Page </a>
</body>
<!-- here second.html will be searched in the current directory-->		​
<a href="../second.html"> Click for second page </a>
<!-- here second.html will be searched in the parent directory-->
<a href="static/second.html"> Click for second page </a>
<!-- here second.html will be searched in the static directory from current directory -->
Create a Bookmark
HTML bookmarks are used to allow users to jump to specific parts of a Web page. Bookmarks can be useful if your web page is very long. To make a bookmark, you must first create the bookmark, and then add a link to it. When the link is clicked, the page will scroll to the location with the bookmark. To create a bookmark href value should be #bookmakname
HTMLOutput
<h2 id="tips"> Useful Tips Section </h2>Create a bookmark with the id attribute
<a href="#tips"> Visit the useful tips section </a>
<!-- note here use of # symbol -->		Add a link to the bookmark ("Useful Tips Section"), from within the same page

Handling Images
The <img> tag is used to insert an image into an HTML document. The image itself isn’t inserted directly into the document, the browser inserts an HTML image from the source specified in the <img> tag.
There are few important attributes for an <img> element : 
  1. src (source) : shows the image source. It is required, as it defines the path to the image. The value of the src attribute can be either the file name it is in the same directory or path if in the different directory or its URL if it is available as internet resource.
  2. alt : defines an alternate name for the image. Its value is a descriptive text displayed in the browser if the image can not be loaded or image can not be found in the given location.
  3. height and width attributes : can be used for specifying the respective dimension of the image in pixels.
HTML   Output
<!-- image file in the same folder as that of html page -->
<img src="Hydrangeas.jpg" width="200" height="200" />		​
<!-- image file in the images folder of current directory -->
<img src="images/Hydrangeas.jpg" width="200" height="200" />
Image as hyperlink 
To make HTML images clickable, you should place the <img> tag inside the <a> tag, which is used for inserting an HTML image link. It is possible to click the image and jump to the destination page.
  <a href="formatting.html">
        <img src="Hydrangeas.jpg" width="200" height="200" />
  </a>
Note : The <img> tag is empty, which means that the explicit closing tag isn’t required. It contains only attributes.
